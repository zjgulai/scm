# 麦肯锡消费者分析图表 Skill

## 概述

消费者分析是商业洞察的核心，通过系统化的图表方法，能够深入理解用户行为、识别增长机会、优化运营策略。

## 十大消费者分析图表类型

### 1. 消费者构成变化图 (Consumer Composition Change)

**图表类型**: 堆叠柱状图

**核心结论**:
- 用户健康度诊断
- 流失/新进/留存趋势
- 用户结构变化

**适用场景**:
```
✓ 用户健康度分析
✓ 新老客结构分析
✓ 流失预警分析
✓ 用户增长归因

✗ 不适合: 单一时间点分析
```

**结构示例**:
```
消费者构成变化

100% ┤ ████████████████████████████
     │ ▓▓▓▓▓▓▓▓▓▓██████████████████
 80% │ ░░░░░░░░░░▓▓▓▓▓▓▓▓▓▓████████
     │ ████████░░░░░░░░░░▓▓▓▓▓▓▓▓▓▓
 60% │ ████████████████░░░░░░░░░░▓▓
     │ ████████████████████████░░░░
 40% │ ████████████████████████████
     └────────────────────────────
       MAT21    MAT22    MAT23

  █重复买者  ▓新进买者  ░流失买者

结论: 流失买者占比从20%升至35%，用户健康度下降
```

**母婴电商案例**:
```markdown
# 流失买者占比从22%升至32%，用户健康度显著下降

## 通过图表可知
• 重复买者占比从65%降至55%，-10pp
• 新进买者占比从13%降至13%，持平
• 流失买者占比从22%升至32%，+10pp
• 流失买者中45%流向高端竞品，说明产品竞争力不足

## 建议行动
• 短期: 针对流失风险用户启动挽回计划
• 中期: 分析流失原因，优化产品和服务
• 长期: 建立用户健康度监控体系

## 关键词
User Health | Churn Rate | Retention Analysis
```

```markdown
• 拓展: 空巢家庭，开发适合产品
```

### 3. 重要性-表现分析图 (Importance-Performance Analysis)

**图表类型**: 散点图 (IPA矩阵)

**核心结论**:
- 改进优先级识别
- 优势/劣势因素定位
- 资源配置建议

**适用场景**:
```
✓ 流失原因分析
✓ 新客吸引因素分析
✓ 产品改进优先级
✓ 服务优化分析

✗ 不适合: 单一因素分析
```

**结构示例**:
```
重要性-表现分析 (IPA矩阵)

重要性
  ↑
  │         ● 产品成分 (高重要/低表现)
高 │    ● 广告提醒
  │              ● 产品美味 (高重要/高表现)
  │         ● 价格促销
  │    ● 包装设计
低 │              ● 渠道便捷
  └────────────────────→ 表现
     低              高

  右上(优势区): 继续保持
  左上(改进区): 优先改进
  右下(维持区): 适当关注
  左下(次要区): 资源可转移

结论: 产品成分和广告提醒是优先改进项
```

**母婴电商案例**:
```markdown
# 流失买者对产品成分(8.5分)和广告提醒(7.8分)重视但卡士表现不足

## 通过图表可知
• 改进区(高重要/低表现):
  - 产品成分: 重要性8.5, 表现6.2, 差距2.3
  - 广告提醒: 重要性7.8, 表现5.5, 差距2.3
  - 包装设计: 重要性6.5, 表现5.0, 差距1.5

• 优势区(高重要/高表现):
  - 产品美味: 重要性8.0, 表现8.5, 超预期
  - 品质认可: 重要性7.5, 表现7.8, 良好

## 改进优先级
1. 产品成分沟通 (最高优先级)
2. 广告投放频次
3. 包装设计升级

## 关键词
IPA Matrix | Priority Analysis | Gap Analysis
```

### 4. 流失流向分析图 (Churn Flow Analysis)

**图表类型**: 双向条形图 / 桑基图

**核心结论**:
- 流失目标识别
- 竞争威胁分析
- 挽回策略方向

**适用场景**:
```
✓ 品牌竞争分析
✓ 用户流失分析
✓ 市场份额变化归因
✓ 竞品对标分析

✗ 不适合: 单一品牌分析
```

**结构示例**:
```
流失流向分析

获得 ↑
    │ ████████████████████ ← 从简爱获得 120万
    │ ████████████████     ← 从君乐宝获得 80万
    │ ████████             ← 从光明获得 50万
────┼──────────────────────────────────────
    │               ▓▓▓▓▓▓▓▓▓▓▓▓▓▓ → 流失给简爱 200万
    │               ▓▓▓▓▓▓▓▓▓▓ → 流失给君乐宝 150万
    │               ▓▓▓▓▓▓ → 流失给光明 80万
    ↓ 流失

    █ 获得  ▓ 流失  净值 = -180万

结论: 净流失给简爱80万，是最大竞争威胁
```

**母婴电商案例**:
```markdown
# 净流失给简爱80万，主要流向其无糖产品线，需加强无糖布局

## 通过图表可知
• 流失分析:
  - 流失给简爱: 200万 (最大)
  - 流失给君乐宝: 150万
  - 流失给光明: 80万

• 获得分析:
  - 从简爱获得: 120万
  - 从君乐宝获得: 80万
  - 从光明获得: 50万

• 净流失:
  - 净流失给简爱: 80万 (最高优先级)
  - 净流失给君乐宝: 70万
  - 净流失给光明: 30万

## 竞争策略
• 防御: 加速无糖产品线开发，抵御简爱
• 进攻: 针对君乐宝用户推广高端产品
• 巩固: 强化与光明用户的差异化定位

## 关键词
Churn Flow | Competitive Analysis | User Migration
```


**结构示例**:
```
购买驱动因素对比

   │  ╱                    ╲

   │    ╲                  ╱
```

```
✓ LTV分析
✓ 用户价值评估
✓ 获客成本决策
✓ 留存投资决策

✗ 不适合: 单一时间点
```

**结构示例**:
```
用户生命周期价值

价值 ┤     ●────●────●────●
     │    ╱
     │   ╱    ●────●────●
     │  ╱
     │ ╱         ●────●
     │╱
     └────────────────────→ 月份
        1    3    6    12   24

     ● 高价值  ● 中价值  ● 低价值

结论: 高价值用户24个月LTV是低价值用户5倍
```

**母婴电商案例**:
```markdown
# 高价值用户24个月LTV达$2500，是低价值用户($480)的5倍

## 通过图表可知
• LTV分层:
  - 高价值用户: $2500 (10%用户)
  - 中价值用户: $800 (30%用户)
  - 低价值用户: $480 (40%用户)
  - 流失风险: $100 (20%用户)

• 关键发现:
  - 10%高价值用户贡献 > 50%营收
  - 中价值用户是最大提升机会
  - 流失风险用户占比20%

## 运营策略
• 高价值: VIP服务、专属权益、提升复购
• 中价值: 升级引导、交叉销售
• 低价值: 激活运营、价格敏感产品
• 流失风险: 挽回优惠、调研反馈

## 关键词
Customer LTV | User Value | Segmentation Strategy
```

## 图表选择决策框架

### 按分析目的选择

| 分析目的 | 推荐图表 | 核心结论类型 |
|---------|---------|------------|
| 用户健康度诊断 | 消费者构成变化图 | 流失率变化趋势 |
| 人群定位对比 | 人群画像对比图 | 品牌人群差异 |
| 改进优先级 | 重要性-表现分析图 | 优先改进项 |
| 竞争威胁分析 | 流失流向分析图 | 主要流失目标 |
| 产品人群定位 | 系列贡献人群分析图 | 产品主力人群 |
| 竞争优劣势 | 购买驱动因素对比图 | 优劣势因素 |
| 场景拓展机会 | 消费场景分析图 | 场景渗透差距 |
| 人群价值评估 | 渗透率-频次矩阵图 | 核心价值人群 |
| 用户价值预测 | 用户生命周期价值图 | LTV分层 |

### 按数据维度选择

| 数据维度 | 推荐图表 |
|---------|---------|
| 时间+构成 | 堆叠柱状图 |
| 品牌+人群 | 堆叠条形图 |
| 重要性+表现 | 散点图(IPA) |
| 流入+流出 | 双向条形图 |
| 品牌+因素 | 折线图 |
| 人群+指标 | 气泡图 |
