# 复杂指标贡献度分析 (Complex Contribution Analysis)

## 概述

在跨境电商中，核心指标往往由多个过程指标通过复杂的四则运算关系构成。例如：

```
运营毛利率 = 1 - 成本率 - 广告费率 - 履约费率
ROI = (销售额 - 成本) / 成本
毛利额 = 访客数 × 转化率 × 客单价 × 毛利率
净利润 = (销售额 × 毛利率 - 固定成本) × (1 - 税率)
```

本Skill提供基于**偏微分**和**全微分**的系统化方法，精确计算各过程指标对结果指标的贡献度。

### 核心概念

| 概念 | 定义 | 业务含义 |
|------|------|---------|
| **静态贡献度** | 偏导数值 | 某因素变动1单位，结果变动多少单位 |
| **弹性系数** | (∂Y/∂X) × (X/Y) | 某因素变动1%，结果变动百分之几 |
| **波动贡献度** | 全微分中的分量 | 某因素变动对总变动的绝对贡献 |
| **贡献占比** | 波动贡献度/总变动 | 某因素变动解释了总变动的百分之几 |

## 核心算法

### 1. 偏微分法（静态贡献度）

**原理**: 对多变量函数 Y = f(X₁, X₂, ..., Xₙ)，偏导数 ∂Y/∂Xᵢ 表示在其他因素不变时，Xᵢ 对 Y 的边际影响。

**常见函数的偏导数**:

#### 1.1 加法关系
```
Y = A + B + C
∂Y/∂A = 1
∂Y/∂B = 1
∂Y/∂C = 1

静态贡献度: A、B、C 的单位变动对 Y 的影响相同
```

#### 1.2 减法关系
```
Y = A - B - C
∂Y/∂A = 1
∂Y/∂B = -1
∂Y/∂C = -1

静态贡献度: A 增加使 Y 增加，B、C 增加使 Y 减少
```

#### 1.3 乘法关系
```
Y = A × B × C
∂Y/∂A = B × C
∂Y/∂B = A × C
∂Y/∂C = A × B

静态贡献度: 某因素的贡献度等于其他因素的乘积
```

#### 1.4 除法关系
```
Y = A / B
∂Y/∂A = 1/B
∂Y/∂B = -A/B²

静态贡献度: A 增加 Y 增加，B 增加 Y 减少（非线性）
```

#### 1.5 混合运算（重点）
```
Y = (A × B + C) / D - E

∂Y/∂A = B / D
∂Y/∂B = A / D
∂Y/∂C = 1 / D
∂Y/∂D = -(A × B + C) / D²
∂Y/∂E = -1
```

### 2. 全微分法（波动贡献度）

**原理**: 全微分将函数的微小变化分解为各变量变化的贡献之和。

**公式**:
```
dY = (∂Y/∂X₁)dX₁ + (∂Y/∂X₂)dX₂ + ... + (∂Y/∂Xₙ)dXₙ

离散形式（用于实际计算）:
ΔY ≈ (∂Y/∂X₁) × ΔX₁ + (∂Y/∂X₂) × ΔX₂ + ... + (∂Y/∂Xₙ) × ΔXₙ
```

**波动贡献度计算**:
```
因素i的波动贡献度 = (∂Y/∂Xᵢ) × ΔXᵢ

因素i的贡献占比 = [(∂Y/∂Xᵢ) × ΔXᵢ] / ΔY × 100%
```

### 3. 弹性系数法

**原理**: 弹性系数消除量纲影响，便于跨因素比较。

**公式**:
```
弹性系数 Eᵢ = (∂Y/∂Xᵢ) × (Xᵢ/Y)

含义: Xᵢ 变动 1%，Y 变动 Eᵢ%
```

**弹性系数解释**:
| 弹性值 | 含义 |
|--------|------|
| |E| > 1 | 富有弹性，因素变动对结果影响大 |
| |E| = 1 | 单位弹性 |
| |E| < 1 | 缺乏弹性，因素变动对结果影响小 |
| E > 0 | 正向影响 |
| E < 0 | 负向影响 |

### 4. 交互效应处理

当因素间存在交互作用时，简单的全微分会有残差。

**交互效应计算**:
```
残差 = ΔY实际 - Σ[(∂Y/∂Xᵢ) × ΔXᵢ]

如果残差较大，说明存在显著交互效应，需要进一步分析。
```

**交互效应分配方法**:
1. **按比例分配**: 将残差按各因素贡献比例分配
2. **Shapley值**: 公平分配交互效应
3. **独立标注**: 单独报告交互效应

## 使用方法

### 输入数据要求

**1. 指标关系定义**
```
格式: Y = f(X₁, X₂, ..., Xₙ)

示例:
- 销售额 = 访客数 × 转化率 × 客单价
- 毛利额 = 销售额 × (1 - 成本率) - 固定成本
- ROI = (销售额 - 成本) / 成本
- 运营毛利率 = 1 - 成本率 - 广告费率 - 履约费率
```

**2. 数据格式**
```csv
period,X1,X2,X3,...,Y
上期,100,0.05,50,...,250
本期,120,0.06,52,...,374.4
```

### 分析流程

```
步骤1: 解析指标关系
├── 识别结果指标 Y
├── 识别过程指标 X₁, X₂, ..., Xₙ
├── 解析运算关系（加减乘除）
└── 构建函数表达式

步骤2: 计算偏导数
├── 对每个过程指标 Xᵢ 求偏导 ∂Y/∂Xᵢ
├── 在基期值处计算偏导数值
└── 得到静态贡献度（敏感度）

步骤3: 计算弹性系数
├── 计算每个因素的弹性系数 Eᵢ
├── 排序识别敏感因素
└── 标注正/负向影响

步骤4: 计算波动贡献度
├── 计算各因素变动 ΔXᵢ
├── 计算波动贡献 = (∂Y/∂Xᵢ) × ΔXᵢ
├── 计算贡献占比
└── 验证: Σ贡献 ≈ ΔY

步骤5: 处理交互效应
├── 计算残差
├── 判断是否显著
└── 分配或单独报告

步骤6: 输出报告
├── 静态贡献度表
├── 波动贡献度表
├── 弹性系数排序
└── 优化建议
```

## 核心公式库

### 常见指标关系的偏导数

#### 公式1: 乘法链式 Y = A × B × C
```
∂Y/∂A = B × C
∂Y/∂B = A × C
∂Y/∂C = A × B

弹性: E_A = E_B = E_C = 1

波动贡献:
ΔY_A = (B₀ × C₀) × ΔA
ΔY_B = (A₁ × C₀) × ΔB  (链式替代)
或 ΔY_B = (A₀ × C₀) × ΔB  (同时替代)
```

#### 公式2: 减法链式 Y = 1 - A - B - C
```
∂Y/∂A = -1
∂Y/∂B = -1
∂Y/∂C = -1

波动贡献:
ΔY_A = -ΔA
ΔY_B = -ΔB
ΔY_C = -ΔC
```

#### 公式3: 混合型 Y = (A × B) / C
```
∂Y/∂A = B / C
∂Y/∂B = A / C
∂Y/∂C = -(A × B) / C²

波动贡献:
ΔY_A = (B₀/C₀) × ΔA
ΔY_B = (A₁/C₀) × ΔB
ΔY_C = -(A₁×B₁/C₀²) × ΔC
```

#### 公式4: ROI型 Y = (A - B) / B = A/B - 1
```
∂Y/∂A = 1/B
∂Y/∂B = -A/B²

波动贡献:
ΔY_A = (1/B₀) × ΔA
ΔY_B = -(A₀/B₀²) × ΔB
```

#### 公式5: 毛利型 Y = A × (1 - B) - C
```
∂Y/∂A = 1 - B
∂Y/∂B = -A
∂Y/∂C = -1

波动贡献:
ΔY_A = (1-B₀) × ΔA
ΔY_B = (-A₁) × ΔB
ΔY_C = -ΔC
```

#### 公式6: 复合型 Y = (A × B + C × D) / E
```
∂Y/∂A = B/E
∂Y/∂B = A/E
∂Y/∂C = D/E
∂Y/∂D = C/E
∂Y/∂E = -(A×B + C×D)/E²
```

#### 公式7: 幂次型 Y = A^α × B^β
```
∂Y/∂A = α × A^(α-1) × B^β
∂Y/∂B = β × A^α × B^(β-1)

弹性: E_A = α, E_B = β
```

#### 公式8: 对数型 Y = A × ln(B)
```
∂Y/∂A = ln(B)
∂Y/∂B = A/B
```

## 示例场景

### 示例1: ROI 变动归因

**场景**: 跨境电商ROI从2.5下降到2.0，需要归因

**指标关系**:
```
ROI = (销售额 - 成本) / 成本 = 销售额/成本 - 1
```

**数据**:
| 指标 | 上期 | 本期 | 变动 |
|------|------|------|------|
| 销售额 | 250万 | 280万 | +30万 |
| 成本 | 100万 | 140万 | +40万 |
| ROI | 2.5 | 1.0 | -1.5 |

**偏导数计算**:
```
设 Y = ROI, A = 销售额, B = 成本
Y = A/B - 1

∂Y/∂A = 1/B
∂Y/∂B = -A/B²

在基期 (A=250, B=100):
∂Y/∂A = 1/100 = 0.01
∂Y/∂B = -250/10000 = -0.025
```

**波动贡献计算**:
```
销售额贡献 = 0.01 × (+30万) = +0.3
成本贡献 = -0.025 × (+40万) = -1.0

一阶全微分预测: ΔY = +0.3 + (-1.0) = -0.7
实际变动: ΔY = 1.0 - 2.5 = -1.5

交互效应: -1.5 - (-0.7) = -0.8
```

**归因结果**:
| 因素 | 波动贡献 | 贡献占比 |
|------|---------|---------|
| 销售额增加 | +0.3 | -20% |
| 成本增加 | -1.0 | 67% |
| 交互效应 | -0.8 | 53% |
| **合计** | **-1.5** | **100%** |

**结论**: 成本上升是ROI下降的主因，同时销售额增长未能抵消成本增幅。

### 示例2: 毛利额多因素归因

**场景**: 分析毛利额变动原因

**指标关系**:
```
毛利额 = 访客数 × 转化率 × 客单价 × 毛利率
```

**数据**:
| 因素 | 上期 | 本期 | 变动 | 变动率 |
|------|------|------|------|--------|
| 访客数 | 10万 | 12万 | +2万 | +20% |
| 转化率 | 5% | 4.5% | -0.5% | -10% |
| 客单价 | 100元 | 110元 | +10元 | +10% |
| 毛利率 | 30% | 28% | -2% | -7% |
| **毛利额** | **15万** | **16.6万** | **+1.6万** | **+11%** |

**偏导数计算** (在基期):
```
Y = A × B × C × D

∂Y/∂A = B × C × D = 0.05 × 100 × 0.3 = 1.5
∂Y/∂B = A × C × D = 10万 × 100 × 0.3 = 300万
∂Y/∂C = A × B × D = 10万 × 0.05 × 0.3 = 1500
∂Y/∂D = A × B × C = 10万 × 0.05 × 100 = 50万
```

**波动贡献计算**:
```
访客数贡献 = 1.5 × (+2万) = +3万
转化率贡献 = 300万 × (-0.5%) = -1.5万
客单价贡献 = 1500 × (+10) = +1.5万
毛利率贡献 = 50万 × (-2%) = -1万

一阶合计 = +3 - 1.5 + 1.5 - 1 = +2万
实际变动 = +1.6万
交互效应 = 1.6 - 2 = -0.4万
```

**归因结果**:
| 因素 | 波动贡献 | 贡献占比 | 弹性系数 |
|------|---------|---------|---------|
| 访客数 | +3万 | 187.5% | 1.0 |
| 转化率 | -1.5万 | -93.8% | 1.0 |
| 客单价 | +1.5万 | 93.8% | 1.0 |
| 毛利率 | -1万 | -62.5% | 1.0 |
| 交互效应 | -0.4万 | -25% | - |
| **合计** | **+1.6万** | **100%** | - |

**结论**:
1. 访客数增长贡献最大（+187.5%）
2. 转化率下降拖累毛利（-93.8%）
3. 各因素弹性系数相同（乘法关系特性）
4. 负交互效应说明多因素同时变动的综合效果小于单独效果之和

### 示例3: 运营毛利率归因

**场景**: 运营毛利率从32%下降到30%

**指标关系**:
```
运营毛利率 = 1 - 成本率 - 广告费率 - 履约费率
```

**数据**:
| 因素 | 上期 | 本期 | 变动 |
|------|------|------|------|
| 成本率 | 45% | 46% | +1% |
| 广告费率 | 15% | 16% | +1% |
| 履约费率 | 8% | 8% | 0% |
| **运营毛利率** | **32%** | **30%** | **-2%** |

**偏导数**:
```
∂Y/∂成本率 = -1
∂Y/∂广告费率 = -1
∂Y/∂履约费率 = -1
```

**波动贡献**:
```
成本率贡献 = -1 × (+1%) = -1%
广告费率贡献 = -1 × (+1%) = -1%
履约费率贡献 = -1 × (0%) = 0%

合计 = -2% = 实际变动 ✓
```

**归因结果**:
| 因素 | 波动贡献 | 贡献占比 |
|------|---------|---------|
| 成本率 | -1% | 50% |
| 广告费率 | -1% | 50% |
| 履约费率 | 0% | 0% |

**结论**: 成本率和广告费率各贡献50%，需同时优化。

## 注意事项

### 方法选择

| 场景 | 推荐方法 | 原因 |
|------|---------|------|
| 简单加法/减法 | 直接分解 | 无交互效应 |
| 简单乘法 | 链式替代或全微分 | 有小交互效应 |
| 复杂混合运算 | 偏微分+全微分 | 精确计算 |
| 高精度需求 | 二阶泰勒展开 | 捕捉交互效应 |

### 交互效应处理

1. **小交互效应（<10%）**: 可忽略或按比例分配
2. **中等交互效应（10-30%）**: 单独报告，说明来源
3. **大交互效应（>30%）**: 需深入分析，考虑二阶效应

### 计算验证

**必须验证**: 一阶全微分之和 + 交互效应 = 实际变动

```python
# 验证代码示例
def verify_contribution(Y_func, X_base, X_current):
    """
    验证贡献度计算的正确性

    Y_func: 结果指标函数
    X_base: 基期各因素值
    X_current: 本期各因素值
    """
    Y_base = Y_func(*X_base)
    Y_current = Y_func(*X_current)
    actual_delta = Y_current - Y_base

    # 计算一阶贡献
    first_order_sum = sum(contributions)
    interaction = actual_delta - first_order_sum

    return {
        'actual_delta': actual_delta,
        'first_order_sum': first_order_sum,
        'interaction': interaction,
        'is_valid': abs(interaction) < 0.01 * abs(actual_delta)
    }
```

### 常见误区

1. **混淆敏感度和贡献度**
   - 敏感度高不代表贡献度大
   - 需要结合变动幅度计算

2. **忽略交互效应**
   - 多因素同时大幅变动时，交互效应显著
   - 不能简单将一阶贡献等同于总贡献

3. **量纲问题**
   - 使用弹性系数进行跨因素比较
   - 偏导数有量纲，直接比较可能误导
