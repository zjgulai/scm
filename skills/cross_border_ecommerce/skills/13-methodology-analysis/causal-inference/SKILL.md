# 跨境电商因果推断分析

## 一、Skill概述

因果推断是解决"为什么"问题的方法。在无法做随机实验的情况下，通过观察数据推断因果关系，找出关键影响因素。

**核心问题**：X是否导致了Y？

**核心价值**：
- 找出关键影响因素
- 区分相关性与因果性
- 支持科学决策
- 评估干预效果

---

## 二、因果关系识别

### 2.1 因果关系的特征

```
因果关系链：
X → Y

例子：
烦心事 → 没睡好 → 很困
工作多 → 睡太晚 → 困
```

### 2.2 多因素共同影响

```
实际情况往往是多因素共同作用：

周围吵 ─┐
烦心事 ─┼→ 睡不好 → 困
工作多 ─┤
没吃药 ─┘

问题：到底哪个是关键因素？
```

### 2.3 相关性 ≠ 因果性

```
常见误区：
- 广告投入越多，销售额越高 → 可能是销量跌时减少投放
- 烧香金额与GDP高度相关 → 纯粹的数字游戏

判断原则：
- 相关性：X和Y同时变化
- 因果性：X的变化导致Y的变化
```

## 三、因果推断方法

### 3.1 随机实验（金标准）

```
原理：通过随机分组，确保各组特征一致

步骤：
1. 招募实验对象
2. 收集基础属性（确保可比较）
3. 随机分组
4. 对比实验前各属性（双样本T检验不显著）
5. 实施干预
6. 对比结果指标
```

**局限性**：很多场景无法做随机实验

### 3.2 分层法（控制变量）

```
原理：在观察数据中，通过分层控制混杂因素

示例：评估优惠券效果

原始数据：
├── 发券组：转化率15%
└── 未发券组：转化率8%
结论：优惠券有效？

分层后（按用户活跃度分层）：
              发券组    未发券组
高活跃用户    25%       22%
中活跃用户    12%       10%
低活跃用户    5%        4%

结论：各层差异不大，原始差异来自用户结构不同
```

### 3.3 匹配法（PSM）

```
原理：为处理组找到特征相似的对照组

倾向得分匹配（PSM）步骤：
1. 计算每个用户接受处理的概率（倾向得分）
2. 根据倾向得分匹配相似用户
3. 比较匹配后的两组差异

适用场景：
- 评估会员权益效果
- 评估促销活动效果
- 评估营销干预效果
```

### 3.4 差分法（DID）

```
原理：对比处理组和对照组在干预前后的变化差异

双重差分模型：
效应 = (处理组后 - 处理组前) - (对照组后 - 对照组前)

示例：评估某地区促销活动效果
                活动前    活动后    差异
处理地区        100       150      +50
对照地区        80        100      +20
DID效应                            +30

适用场景：
- 政策效果评估
- 活动效果评估
- 改版效果评估
```

### 3.5 断点回归（RDD）

```
原理：利用政策/规则的临界点，比较临界点两侧

示例：评估VIP权益效果
- 消费满1000元成为VIP
- 比较990元和1010元用户的行为差异
- 这两组用户本质上相似，只是VIP身份不同

适用场景：
- 会员等级权益评估
- 门槛类政策效果评估
```

## 四、发现因果关系的方法

### 4.1 第一步：感觉

```
"我觉得"是很重要的起点

示例：
- 我觉得，球星多的球队成绩好
- 我觉得，学历高的人工资高
- 我觉得，买面包的人会顺手买牛奶

能说出"我觉得"，好过连说都说不出来
```

### 4.2 第二步：观察

```
用数据验证"我觉得"

三种观察方式：
1. 指标+事件：事件发生后，指标是否有变化
2. 指标+维度：不同维度下，指标是否有差异
3. 指标+指标：两个指标是否有相关性
```

### 4.3 第三步：推断

```
排除其他可能性，确认因果关系

常见陷阱：
- 反向因果：X导致Y，还是Y导致X？
- 混杂因素：Z同时影响了X和Y
- 选择偏差：观察的样本不具代表性
```

## 五、跨境电商应用场景

### 5.1 广告效果评估

```
问题：投放广告后销量提升，是广告的效果吗？

方法：PSM + DID

步骤：
1. 用PSM找到未投放广告的相似用户/地区
2. 用DID对比投放前后的差异
3. 剔除自然增长，得到广告净效果
```

### 5.2 会员权益评估

```
问题：成为会员后消费增加，是会员权益的效果吗？

方法：RDD（断点回归）

步骤：
1. 找到会员门槛（如消费满$100）
2. 比较$98和$102用户的行为差异
3. 这两组用户本质相似，差异来自会员身份
```

### 5.3 促销活动评估

```
问题：促销期间销量提升，是促销的效果吗？

方法：分层 + 对照

步骤：
1. 找到未参与促销的相似商品/用户
2. 分层对比（控制季节性、商品类型等）
3. 计算促销带来的增量
```

## 七、常见问题与解决方案

### 7.1 样本选择偏差

**问题**：处理组和对照组本身就有差异

**解决**：
- 使用匹配法找到相似对照组
- 使用分层法控制混杂因素
- 使用差分法消除固定差异

### 7.2 遗漏变量偏差

**问题**：存在未观测的混杂因素

**解决**：
- 尽可能收集更多变量
- 使用工具变量法
- 进行敏感性分析

### 7.3 反向因果

**问题**：可能是Y影响了X，而非X影响了Y

**解决**：
- 使用时间序列，确保X发生在Y之前
- 使用工具变量
- 结合业务逻辑判断

## 九、注意事项

1. **因果推断不是背诵公式**：是梳理逻辑、提假设、找方法
2. **明确问题和假设**：先搞清楚要解决什么问题
3. **多种方法交叉验证**：使用多种方法，结果一致才可信
4. **承认局限性**：观察数据的因果推断总有局限
5. **结合业务判断**：数据分析需要结合业务逻辑
