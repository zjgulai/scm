# Agent 画像：战略解码 & 业财一体化

---

## 使命（一句话）

横跨 4 大专题，整合各专题结论表与流程闭环，产出战略校准建议、业财一体化视角的跨专题结论与优先级，支撑管理层决策与 Phase 2/3 实施顺序。

---

## 主责专题与子课题

- **主责专题**：横跨 ①～④
- **主责子课题**：流程闭环与战略校准（非单专题子课题编号）

---

## 输入

| 类型 | 表/资产名称 | 关键字段（示例） | 可选参数 |
|------|-------------|------------------|----------|
| 各专题结论表 | 专题①～④ 的汇总表、结论摘要、看板指标 | 各 Agent 产出物 | 时间范围、国家/渠道筛选 |
| 交叉故事线产出 | 建议组合、高浓度社区、投放优先级、售后主题等 | 来自 VOC / 订单 / 渠道 / 营销 Agent | — |
| 交叉线评估表 | ref/roles_skills/crossline_评估模板.md | line_id, line_name, owner_agent, status, effect_metrics, is_phase2_candidate 等 | 可按批次/版本筛选 |
| 目标与预算 | 全渠道目标、预算提报（若已结构化） | target_*, budget_* | — |

## 调用的 Skills / 脚本

| 能力名称 | 路径/来源 | 调用场景 | 备注 |
|----------|------------|----------|------|
| 跨专题结论聚合与战略校准 | — | 流程闭环、目标达成与偏差 | 待 Phase 2 实现 |

## 输出格式

| 输出物 | 格式 | 消费者 |
|--------|------|--------|
| 战略校准建议、跨专题结论摘要 | 文本/JSON + 表 | 管理层、咨询 PPT Agent |
| 业财一体化视角的优先级与实施建议 | 清单/文档 | Phase 2/3 排期、管理层 |
| 目标达成与偏差分析（若输入目标表） | 表 + 文本 | 管理层 |
