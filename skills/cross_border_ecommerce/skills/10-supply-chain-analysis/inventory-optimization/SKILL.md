# 母婴跨境电商库存优化分析 Skill

## 概述

本Skill帮助母婴跨境电商卖家优化库存管理，在避免断货和降低库存成本之间找到平衡。通过科学的库存模型（安全库存、EOQ、ABC分类），提高库存周转效率，减少资金占用。

### 母婴库存管理特殊性

母婴产品库存管理具有独特特征:
- **保质期敏感**: 奶粉、辅食等产品对保质期要求高，消费者偏好新鲜产品
- **断货损失大**: 母婴客户断货会导致客户流失和信任损失
- **批次追溯要求**: 配方奶粉等强监管产品需要批次追溯能力
- **季节性明显**: 服装、节日礼品等季节性波动大
- **订阅制保障**: 订阅客户需要稳定的库存保障
- **安全库存要求高**: 母婴客户对产品可得性要求高

本Skill充分考虑这些特性，提供母婴专属的库存管理策略。

## 核心功能

### 1. 安全库存计算(母婴增强)

#### 基础安全库存
- **安全库存(SS)**: 应对需求波动和供应延迟的缓冲库存
- **计算公式**: SS = Z × σd × √L
- **服务水平配置**: 母婴行业建议95-99%服务水平
- **动态调整**: 基于季节性、订阅客户比例调整

#### 母婴安全库存调整因子

| 因素 | 调整系数 | 说明 |
|------|---------|------|
| **高订阅客户比例(>30%)** | × 1.1 | 订阅客户需求稳定，可适当降低 |
| **高频刚需品类(纸尿裤/奶粉)** | × 1.2 | 断货损失大，需要更高安全库存 |
| **长保质期产品(>18个月)** | × 1.0 | 正常安全库存 |
| **短保质期产品(6-12个月)** | × 0.9 | 避免效期损失，适度降低 |
| **季节性产品** | × 0.8-1.3 | 旺季×1.3，淡季×0.8 |
| **节日产品** | × 1.5 | 节日前2个月需要高安全库存 |

#### 母婴安全库存计算示例

```
品类: 纸尿裤(高频刚需)
- 基础安全库存: 100包
- 高频刚需调整: × 1.2 = 120包
- 订阅客户比例30%: × 0.9 = 108包
- 最终安全库存: 110包

品类: 配方奶粉(保质期敏感)
- 基础安全库存: 80罐
- 高频刚需调整: × 1.2 = 96罐
- 保质期调整(12个月): × 0.9 = 86罐
- 最终安全库存: 85罐

品类: 玩具(季节性+节日)
- 基础安全库存: 50件
- 节日前调整(圣诞): × 1.5 = 75件
- 最终安全库存: 75件
```

### 2. 保质期管理(母婴专属)

#### 保质期管理框架

```
母婴产品保质期管理:

入库要求:
- 生产日期 < 2个月 (奶粉、辅食)
- 生产日期 < 3个月 (其他产品)
- 剩余保质期 > 18个月 (奶粉)
- 剩余保质期 > 12个月 (辅食)

库存管理:
- 先进先出(FIFO)原则
- 批次管理(每批独立管理)
- 效期预警系统
- 临期产品处理机制

出库要求:
- 优先出库临期产品(在保质期合格前提下)
- 不出库保质期 < 6个月的产品(奶粉)
- 不出库保质期 < 3个月的产品(辅食)
```

#### 效期预警等级

| 预警等级 | 剩余保质期 | 处理措施 | 折扣建议 |
|---------|-----------|---------|---------|
| 🟢 正常 | >12个月 | 正常销售 | 无折扣 |
| 🟡 关注 | 9-12个月 | 加快周转 | 5-8%折扣 |
| 🟠 预警 | 6-9个月 | 促销推动 | 10-15%折扣 |
| 🔴 临期 | 3-6个月 | 清仓处理 | 20-30%折扣 |
| ⛔ 禁售 | <3个月 | 下架销毁 | 员工内购/捐赠 |

#### 效期损失计算

```
效期损失 = 临期产品数量 × (成本 - 清仓价)

示例(奶粉):
- 临期产品: 50罐
- 成本: $18/罐
- 正常售价: $29.99
- 清仓价(20%折扣): $23.99
- 效期损失: 50 × ($18 × 20%) = $180
- 如果不清仓(销毁): 50 × $18 = $900

结论: 及时清仓减少损失$720
```

### 4. 经济订货量(EOQ)(母婴增强)

## 核心功能

### 1. 安全库存计算
- **安全库存(SS)**: 应对需求波动和供应延迟的缓冲库存
- **计算公式**: SS = Z × σd × √L
- **服务水平配置**: 90%/95%/99%对应不同Z值
- **动态调整**: 基于季节性和趋势调整

### 2. 经济订货量(EOQ)
- **EOQ模型**: 平衡订货成本和持有成本
- **计算公式**: Q* = √(2DS/H)
- **扩展模型**: 考虑数量折扣、允许缺货
- **敏感性分析**: 参数变化对EOQ的影响

### 3. 库存周转分析
- **周转率(ITR)**: 销售速度与库存的比值
- **周转天数(DOS)**: 平均库存可销售天数
- **趋势分析**: 同比/环比周转变化
- **对标分析**: 行业基准对比

### 4. ABC库存分类
- **A类**: 高价值、低数量（重点管理）
- **B类**: 中价值、中数量（常规管理）
- **C类**: 低价值、高数量（简化管理）
- **差异化策略**: 不同类别采用不同管理策略

### 5. 补货时机预测
- **再订货点(ROP)**: ROP = 日均销量 × 交货期 + 安全库存
- **补货预警**: 库存低于ROP时触发
- **补货建议**: 建议订货量和时间

## 核心公式

### 安全库存(SS)
```
SS = Z × σd × √L

其中:
- Z: 服务水平对应的Z值
  - 90%: Z = 1.28
  - 95%: Z = 1.65
  - 99%: Z = 2.33
- σd: 日需求标准差
- L: 交货期（天）

示例:
- 服务水平: 95% → Z = 1.65
- 日需求标准差: 15件
- 交货期: 21天
- SS = 1.65 × 15 × √21 = 113件
```

### 再订货点(ROP)
```
ROP = d̄ × L + SS

其中:
- d̄: 日均需求量
- L: 交货期
- SS: 安全库存

示例:
- 日均销量: 100件
- 交货期: 14天
- 安全库存: 80件
- ROP = 100 × 14 + 80 = 1480件
```

### 经济订货量(EOQ)
```
Q* = √(2DS/H)

其中:
- D: 年需求量
- S: 每次订货成本
- H: 单位年持有成本 = 单位成本 × 持有成本率

年总成本 = (D/Q) × S + (Q/2) × H

示例:
- 年需求量: 10,000件
- 订货成本: $50/次
- 单位成本: $20
- 持有成本率: 25%
- H = 20 × 25% = $5
- Q* = √(2 × 10000 × 50 / 5) = 447件
```

### 库存周转率(ITR)
```
ITR = 年销售成本 / 平均库存金额

周转天数 = 365 / ITR

示例:
- 年销售成本: $1,000,000
- 平均库存: $200,000
- ITR = 1000000 / 200000 = 5次/年
- 周转天数 = 365 / 5 = 73天
```

### ABC分类
```
1. 按年销售金额降序排列所有SKU
2. 计算每个SKU的销售金额
3. 计算累计金额占比
4. 按累计占比划分:
   - A类: 0-80%
   - B类: 80-95%
   - C类: 95-100%
```

## 示例

### 示例1: 新品安全库存设定

**场景**: 新产品上架，需要设定安全库存

**输入数据**:
- 预测日均销量: 30件
- 销量波动估计: ±50%（新品不确定性大）
- 交货期: 30天（跨境物流）
- 服务水平目标: 95%

**计算过程**:
1. 销量标准差估计: 30 × 50% = 15件
2. 安全库存: SS = 1.65 × 15 × √30 = 136件
3. 再订货点: ROP = 30 × 30 + 136 = 1036件

**建议**:
- 初始安全库存: 150件（考虑新品不确定性，+10%）
- 再订货点: 1050件
- 首次订货量: 预测30天销量 + 安全库存 = 1050件

### 示例2: 季节性产品库存规划

**场景**: 旺季产品Q4备货

**输入数据**:
- Q4预测销量: 5,000件
- 淡季月销量: 500件
- 交货期: 45天
- 仓储成本高

**分析**:
1. 旺季日均销量: 5000 / 90 = 56件
2. 考虑季节性，安全库存增加50%
3. 标准安全库存: 1.65 × 15 × √45 = 166件
4. 调整后安全库存: 166 × 1.5 = 249件

**备货计划**:
- 旺季前60天开始备货
- 分批到货，避免一次性库存压力
- 旺季结束前30天停止补货

### 示例3: 多SKU库存优化

## 注意事项

### 交货期准确性
- 跨境物流交货期波动大
- 需要考虑清关时间
- 建议使用历史实际交货期

### 季节性调整
- 旺季前提前增加安全库存
- 淡季适当降低库存水平
- 促销活动前特殊规划

### 供应商可靠性
- 供应商交货准时率影响安全库存
- 新供应商需要更保守的策略
- 建立供应商评分机制

### 资金约束
- 安全库存需要资金支持
- 需要平衡服务水平与资金成本
- A类产品优先保障

### 多仓库库存优化

```python
# 考虑多仓库的安全库存分配
def allocate_safety_stock(total_ss, warehouses):
    """
    按需求比例分配安全库存到各仓库
    """
    total_demand = sum(w['demand'] for w in warehouses)

    for warehouse in warehouses:
        allocation_ratio = warehouse['demand'] / total_demand
        warehouse['ss'] = total_ss * allocation_ratio

    return warehouses
```

# 基于滚动预测的动态安全库存
def dynamic_safety_stock(sales_history, lead_time, service_level):
    """
    使用最近30天数据动态计算安全库存
    """
    # 滚动窗口计算
    recent_std = sales_history[-30:].std()
