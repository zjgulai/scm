# 专题一：分析数据总表 — 分析计划

## 一、数据与口径

- **数据源**：`data_example/专题一：分析数据总表.xlsx`，Sheet = `①`
- **维度**：平台、区域、月份
- **指标**：销售额、品线毛利额、品线毛利率（其中 品线毛利率 = 品线毛利额 / 销售额，表中已存在则直接使用，聚合时按 毛利额/销售额 重算）
- **时间窗口**：
  - **MAT2026P1**：2025-02 ～ 2026-01
  - **MAT2025P1**（同期）：2024-02 ～ 2025-01
  - **YTD2026P1**：2026-01
  - **YTD2025P1**（同期）：2025-01

## 二、分析步骤

### 0. 描述性统计

- 对 销售额、品线毛利额、品线毛利率 做描述性统计（条数、均值、标准差、 min/max、分位数等）
- 按 平台、区域 的分布与样本量

### 1. 整体指标（步骤1）

- 计算 **整体** 在四个时间窗口的：销售额、品线毛利额、品线毛利率（品线毛利率 = 汇总毛利额/汇总销售额）
- 计算 **MAT2026P1** 相对 **MAT2025P1** 的：同比差值、同比率（销售额、品线毛利额、品线毛利率）
- 计算 **YTD2026P1** 相对 **YTD2025P1** 的：同比差值、同比率

### 2. 分平台×区域聚合表（步骤2）

- 按 平台×区域 分别计算上述四个时间窗口的 销售额、品线毛利额、品线毛利率
- 每个 平台×区域 计算 MAT 同比差值/同比率、YTD 同比差值/同比率
- 合并为一张总表，保存为 **新 Excel 文件**（第一个 Sheet 为该聚合表）

### 3. 平台归因（步骤3）

- **各平台对整体毛利额的贡献占比** = 该平台毛利额 / 整体毛利额（分时间窗口）
- **各平台对整体毛利率的贡献占比** = (该平台毛利额/总销售额) / (总毛利额/总销售额) = 该平台毛利额/总毛利额（与毛利额贡献占比一致，分时间窗口）

### 4. 区域归因（步骤4）

- **各区域对整体毛利额的贡献占比** = 该区域毛利额 / 整体毛利额
- **各区域对整体毛利率的贡献占比** = 该区域毛利额 / 总毛利额
- 分时间窗口输出

### 5. 平台毛利率同比波动对整体毛利率波动的贡献（步骤5）

- **整体毛利率波动** = 当期整体毛利率 − 同期整体毛利率（MAT 与 YTD 分别算）
- **某平台贡献** = 当期该平台销售额占比 × (该平台当期毛利率 − 该平台同期毛利率)
- **贡献率** = 该贡献值（带符号）
- **贡献占比** = 该贡献 / 整体毛利率波动（整体波动≠0 时）
- **正/负贡献**：贡献 > 0 为正，< 0 为负

### 6. 平台×区域毛利率同比波动贡献（步骤6）

- 与步骤5相同逻辑，主体改为 **平台×区域**
- 某 平台×区域 贡献 = 当期该组合销售额占比 × (该组合当期毛利率 − 该组合同期毛利率)
- 输出贡献率、贡献占比、正/负标注

### 7. 结果写入新 Excel（步骤7）

- Sheet1：步骤2 的 分平台×区域 聚合表
- Sheet2：步骤3 平台归因
- Sheet3：步骤4 区域归因
- Sheet4：步骤5 平台毛利率波动贡献
- Sheet5：步骤6 平台×区域毛利率波动贡献
- 保证数据与上述公式一致、可复核

### 8. 归因分析结论（步骤8）

从以下三方面输出文字结论：

1. **MAT2026P1 与 YTD2026P1** 两个时间窗口的 **整体毛利额** 主要贡献平台及占比；相对 MAT2025P1/YTD2025P1，各平台贡献占比的变化。
2. **MAT2026P1 与 YTD2026P1** 两个时间窗口的 **整体毛利率同比波动** 主要贡献平台及占比；各平台下哪些区域正贡献/负贡献占比较大、需重点关注。
3. 其他有洞察的结论（如区域结构、平台集中度、毛利率与规模关系等）。
