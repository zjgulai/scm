# 跨境电商趋势分析 Skill

## 概述

趋势分析是业务监控和决策的基础。通过系统化的趋势分析，企业可以：
- **追踪业务表现**: 了解关键指标的变化趋势
- **识别季节性模式**: 发现业务的周期性规律
- **预警异常情况**: 及时发现趋势偏离
- **支持预测决策**: 为未来规划提供依据

本Skill提供全面的趋势分析工具，包括同比、环比、MAT、YTD等经典方法。

## 核心功能

### 1. 同比增长率 (YoY - Year over Year)

**定义**: 与去年同期相比的增长率

**公式**:
```
YoY增长率 = (本期值 - 去年同期值) / 去年同期值 × 100%
```

**适用场景**:
- 消除季节性影响
- 年度业绩评估
- 长期趋势判断

**示例**:
```
2024年3月销售额: $120,000
2023年3月销售额: $100,000
YoY增长率 = (120,000 - 100,000) / 100,000 = +20%
```

### 2. 环比增长率 (MoM - Month over Month)

**定义**: 与上一期相比的增长率

**公式**:
```
MoM增长率 = (本期值 - 上期值) / 上期值 × 100%
```

**适用场景**:
- 短期波动监控
- 运营效果评估
- 快速响应调整

**示例**:
```
3月销售额: $120,000
2月销售额: $110,000
MoM增长率 = (120,000 - 110,000) / 110,000 = +9.1%
```

### 3. 复合增长率 (CAGR)

**定义**: 多期平均年化增长率

**公式**:
```
CAGR = (期末值/期初值)^(1/年数) - 1
```

**适用场景**:
- 长期增长评估
- 投资回报计算
- 业务规划

**示例**:
```
2021年销售额: $500,000
2024年销售额: $800,000
CAGR = (800,000/500,000)^(1/3) - 1 = 17.1%
```

### 4. 滚动累计分析

**MAT (Moving Annual Total)**: 滚动12个月累计
```
2024年3月MAT = 2023年4月~2024年3月累计
```

**YTD (Year to Date)**: 年初至今累计
```
2024年3月YTD = 2024年1月~3月累计
```

**MTD (Month to Date)**: 月初至今累计

**QTD (Quarter to Date)**: 季度初至今累计

### 5. 季节性分解

将时间序列分解为:
```
原始序列 = 趋势项 + 季节性项 + 残差项
```

**趋势项**: 长期发展方向
**季节性项**: 周期性波动（周、月、年）
**残差项**: 随机波动

### 6. 异常点检测

识别与趋势显著偏离的数据点:
- 基于统计方法（Z-score、IQR）
- 基于业务规则（超出预期范围）

## 使用方法

### 输入数据要求

**时间序列数据格式**:
```csv
date,sales,orders,visitors,conversion_rate
2023-01-01,50000,1000,20000,0.05
2023-02-01,55000,1100,22000,0.05
2023-03-01,60000,1200,24000,0.05
...
```

**字段说明**:
| 字段 | 说明 | 格式 |
|------|------|------|
| date | 日期 | YYYY-MM-DD |
| sales | 销售额 | 数值 |
| orders | 订单数 | 整数 |
| visitors | 访客数 | 整数 |
| conversion_rate | 转化率 | 小数 |

```

步骤2: 计算增长率
- 计算YoY增长率
- 计算MoM增长率
- 计算CAGR

步骤3: 滚动累计
- 计算MAT
- 计算YTD
- 计算移动平均

步骤4: 季节性分解
- 识别季节性模式
- 分解趋势和季节项
- 分析残差

步骤5: 异常检测
- 识别异常点
- 分析异常原因
- 生成预警

步骤6: 可视化报告
- 趋势图
- 增长率图
- 季节性分解图
```

## 算法说明

### 移动平均计算

**简单移动平均 (SMA)**:
```
SMA_n = (X₁ + X₂ + ... + Xₙ) / n

例: 7日移动平均
SMA_7 = (D1+D2+D3+D4+D5+D6+D7) / 7
```

**指数移动平均 (EMA)**:
```
EMA_t = α × X_t + (1-α) × EMA_{t-1}

其中 α = 2/(n+1)

EMA对近期数据赋予更高权重，反应更灵敏
```

### 季节性分解算法

**加法模型**:
```
Y(t) = T(t) + S(t) + R(t)

Y: 原始序列
T: 趋势项
S: 季节性项
R: 残差项
```

**乘法模型**:
```
Y(t) = T(t) × S(t) × R(t)

适用于季节性幅度随趋势变化的情况
```

### 异常点检测

**Z-Score方法**:
```
Z = (X - μ) / σ

|Z| > 3 视为异常
```

**IQR方法**:
```
Q1 = 第25百分位数
Q3 = 第75百分位数
IQR = Q3 - Q1

异常值: < Q1 - 1.5×IQR 或 > Q3 + 1.5×IQR
```

## 示例场景

### 示例1: 月度业绩复盘

**数据**: 过去12个月销售数据

**分析**:
```
月份    销售额    MoM     YoY
2023-01  $80K     -      -10%
2023-02  $85K    +6.3%   -5%
2023-03  $90K    +5.9%   0%
...
2024-01  $100K   +8.7%   +25%
2024-02  $110K  +10.0%   +29%
2024-03  $120K   +9.1%   +20%

趋势: 持续增长，YoY保持20%以上
```

**结论**:
- 业务处于健康增长通道
- 近3个月MoM加速增长
- YoY增长稳健，目标达成

### 示例2: 季节性分析

**场景**: 分析年度销售季节性模式

**数据**: 3年历史数据

**发现**:
```
季度销售占比:
Q1: 18% (淡季，春节影响)
Q2: 22% (平稳增长)
Q3: 23% (平稳增长)
Q4: 37% (旺季，黑五圣诞)

月度峰值:
- 11月: 黑五促销，销量+50%
- 12月: 圣诞季，销量+40%
- 2月: 春节淡季，销量-30%
```

**应用**:
- Q3开始备货，准备Q4旺季
- Q1控制库存，减少资金占用
- 11-12月加大广告投入

### 示例3: 多指标关联分析

**场景**: 分析销售额、访客数、转化率的关系

**数据**:
| 月份 | 销售额 | 访客数 | 转化率 |
|------|--------|--------|--------|
| 1月 | $100K | 20K | 5% |
| 2月 | $120K | 24K | 5% |
| 3月 | $132K | 24K | 5.5% |

**归因**:
```
1月→2月: 销售额+20%
- 访客数贡献: (24K-20K) × 5% × $100 = +$20K (100%)
- 转化率贡献: 24K × (5%-5%) × $100 = $0

2月→3月: 销售额+10%
- 访客数贡献: (24K-24K) × 5% × $100 = $0
- 转化率贡献: 24K × (5.5%-5%) × $100 = +$12K (100%)
```

**结论**:
- 1-2月增长由流量驱动
- 2-3月增长由转化率提升驱动
- 需继续优化两方面

## 注意事项

### 数据质量
- 确保数据连续性，避免缺失
- 异常大促需标记，避免误导趋势
- 多币种需统一汇率

### 方法选择
- YoY适合消除季节性
- MoM适合监控短期变化
- MAT适合平滑波动

### 解读技巧
- 关注趋势而非单点
- 结合业务背景分析
- 多指标交叉验证

## 高级功能

### 自动预警

设置阈值自动预警:
```
当YoY增长率 < -10%时，发送预警
当MoM连续3月下降时，发送预警
当实际值偏离预测 > 20%时，发送预警
```

### 趋势预测

基于历史趋势预测未来:
```
使用线性回归或指数平滑
预测未来3-6个月趋势
提供置信区间
```

### 对标分析

与行业或竞品对比:
```
本公司YoY: +20%
行业平均YoY: +15%
表现: 优于行业
```
