# mckinsey-donut-chart Reference

## 触发说明

- Use when users need donut or pie-style composition visuals to communicate share, mix, or structure in a concise executive format.

## 输入建议

- 业务问题: 本页要回答什么管理问题
- 数据口径: 指标定义、时间窗口、分组维度
- 输出场景: 单页图表 / 多页叙事 / 管理层汇报

## 输出建议

- 结论式标题（不是章节标题）
- 3-5 条结论要点（每条有数据支撑）
- 图表或方法选择理由
- 数据来源或关键词

## 快速自检

- 是否回答了原始业务问题
- 图表和结论是否一致
- 信息密度是否可读
- 是否可直接复用到下一页或下一次任务
