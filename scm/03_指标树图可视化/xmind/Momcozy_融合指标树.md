## 综合供应链满意度 (目标: ≥80分)

### 1. 成本满意度维度 (50%权重)

#### 1.1 战略层指标

## 计算公式汇总

### 综合供应链满意度
```
综合供应链满意度 = 成本满意度得分 × 50% + 履约满意度得分 × 50%

成本满意度得分 = 100 - (实际全链路成本率 / 18% × 100)
（成本率越低得分越高，封顶100分，保底0分）

履约满意度得分 = 实际综合履约满意度指数得分
（基于时效40% + 质量40% + 成本效率20%的加权）
```

### 评分等级
| 等级 | 得分区间 | 状态 | 行动建议 |
|------|---------|------|---------|
| A | 90-100 | 卓越 | 保持并寻求优化空间 |
| B | 80-89 | 良好 | 维持现状，关注波动 |
| C | 70-79 | 达标 | 需针对性改进 |
| D | 60-69 | 预警 | 紧急制定改进计划 |
| E | <60 | 危险 | 高层介入，全面复盘 |

## 指标归因机制

### 归因流程
1. **异常触发** → 指标超出阈值（成本±10%，时效-5%，质量+50%）
2. **因素拆解** → 定位问题环节（计划→采购→头程→仓储→配送→客服）
3. **根因分析** → 5Why分析法（预测偏差/执行延迟/资源不足/外部异常/系统故障）
4. **责任判定** → RACI矩阵（R执行/A负责/C咨询/I知情）
5. **改进闭环** → 制定措施→执行→验证→经验沉淀

### 三大归因场景
- **场景1: 大促爆单** → 预测问题/库存问题/履约问题
- **场景2: 滞销积压** → 预测偏差/MOQ问题/销售不力
- **场景3: 退货高峰** → 产品问题/履约问题/客户原因
