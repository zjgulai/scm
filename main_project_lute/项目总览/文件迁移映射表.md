# 文件迁移映射表（旧路径 → 新路径）

## 6）高频引用更新示例

以下是迁移后最常用的新路径写法（用于脚本命令和文档引用）：

- Skills 检索：
  - `python3 skills/cross_border_ecommerce/skills_index.py "关键词"`
  - `python3 skills/mckinsey_ppt/skills_index.py "关键词"`
- 主项目数据脚本：
  - `python main_project_lute/data_example/scripts/...`
- Phase2/3 产出：
  - `main_project_lute/phase2_outputs/...`
  - `main_project_lute/phase3_outputs/...`
- 洞察资料：
  - `knowledge_base/insights/...`
