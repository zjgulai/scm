# Examples: 双引擎知识萃取

## Example A: 《定位》— 消费品品牌团队应用

**Input:**
- 书名：《定位》(Al Ries & Jack Trout)
- 目的：提炼可落地的品牌策略框架
- 场景：消费品品牌团队，正在做品牌升级

### Phase 1 输出骨架

**① 一句话总论：**
品牌竞争的本质不是产品之争，而是消费者心智中"品类第一联想位"的争夺。

**② 核心论点（3 条示例）：**

论点 1：心智定位
- 问题：品牌如何在竞争中胜出？
- 主张：占据消费者心智中的品类第一位置
- 关键逻辑：心智容量有限 → 每品类仅记住 2-3 个品牌 → 先入者优势
- 因果关系：先进入 ≠ 必然占据（需持续强化）
- 前提假设：消费者决策依赖品类联想
- 适用边界：成熟品类有效，全新品类需先建品类认知

**③ 证据地图：**
- 实证数据：少量市场份额数据【证据强度：中】
- 单一案例：大量品牌故事（可口可乐、Avis 等）【强度：低-中】
- 逻辑推演：心智阶梯理论【待验证：缺乏认知科学实证】

**④ 方法清单：**
- 品类阶梯法：定义品类 → 找空位 → 占据
- 对立定位法：找到第一名 → 反向定义自己

**⑤ 反例与局限：**
- 忽略平台型/生态型品牌（如亚马逊横跨多品类）
- 过度泛化"聚焦"，忽略品牌延伸的成功案例

**Phase Gate 判断：**
> 心智定位和品类阶梯法值得进入 Phase 2；部分案例论证因时代局限应搁置。

### Phase 2 输出骨架

**① 模型 1：品类阶梯法**
- 人话：每个品类在消费者脑中只有 3 级台阶，你要抢第一级或者造新梯子。
- 解决：品牌在已有品类中如何找到差异化位置
- 不解决：全新品类从 0 到 1 的需求教育

**② 场景推演：**
- 场景：母婴吸奶器品牌进入北美市场
- 分析路径：现有品类领导者是谁 → 消费者对品类的默认联想 → 是否存在未被占据的心智空位 → 用什么词锚定该空位

**③ 机会与陷阱：**
- ✅ 立即可做：调研目标用户"提到吸奶器首先想到什么"
- ⚠️ 陷阱：把"功能多"当定位（功能不是心智位置）

### Phase 3 输出骨架

**① 第一性原理：**
- 心智稀缺原理：注意力有限，每品类仅存 2-3 个选项

**② Action Table：**

| 原理名称 | 一句话本质 | 最小行动（≤ 5 min） | 避免的错误 |
|----------|-----------|--------------------|-----------| 
| 心智稀缺 | 消费者只记得品类前三 | 每周问 3 个目标用户"提到 X 品类你首先想到谁" | 自以为品牌知名度够高 |

**③ 触发与复盘：**
- 触发条件：当团队讨论"我们的差异化是什么"时 → 先做心智位置调研
- 复盘问题：过去 30 天，目标用户提到品类时是否更多提及我们的品牌名？

### 三句价值判断

1. 最值得带走的是"品类心智位"这个竞争分析视角，它重新定义了竞争对手。
2. 如果只保留一个：品类阶梯法——在做任何品牌决策前先画出目标品类的心智阶梯。
3. 最适合【品牌经理/市场负责人】在【品牌升级或进入新市场】阶段阅读。

## Example B: 长文/报告的快速判断

**Input:**
- 用户：这篇长文我只想知道有没有值得团队采用的方法。

**Expected behavior:**
1. Phase 1 快速执行：重点放在 ③证据地图 和 ⑤反例局限。
2. Phase Gate 严格筛选：只保留有中等以上证据支撑的方法。
3. Phase 2 聚焦：仅输出通过 gate 的 1-2 个模型 + 机会/陷阱。
4. Phase 3 精简：给出 1 条最小行动 + 1 个触发条件即可。
5. 跳过详细场景推演，直接给"采用前提"和"风险提醒"。

## Anti-pattern: 常见错误输出

**错误示范：只写摘要不做判断**

```
这本书讲了定位的重要性，提出了很多有趣的案例，
建议品牌团队认真学习定位理论并应用到实际工作中。
```

问题：
- 没有论证判断（是否成立？）
- 没有证据评估（案例可靠吗？）
- 没有边界标注（什么时候不适用？）
- 行动建议空泛（"认真学习"不可执行）

**错误示范：模型只讲概念不映射场景**

```
定位理论的核心模型是品类阶梯，这个模型很有价值。
```

问题：
- 没有"人话"解释
- 没有适用/不适用边界
- 没有场景推演
- 没有机会点和陷阱
