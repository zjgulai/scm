## 本地重建说明

本节为基于当前项目本地资料重建的仓储与调拨协同执行方案，不等同于钉钉原文复制。重建依据来自 `ref/books/供应链36%方案_Page6节点/11-重点节点04-仓储与调拨协同执行方案.md` 以及本专题 Data / Plan / Report 层。

## 1. 节点定位

仓储与调拨负责让“动作正确”。仓储不是单纯压仓租，调拨不是临时救火；二者共同承担库存健康、现金流效率和跨仓动态平衡。

```text
日清库存异常 -> 周度调拨候选 -> 月度库龄治理 -> 成本率/周转/超龄验收
```

## 2. 执行目标

| 目标 | 管理含义 | 验收指标 |
|---|---|---|
| 控制库龄 | 降低仓储费和跌价风险 | 90天+、180天+库存占比 |
| 保持可售 | 降周转不能以断货为代价 | 现货满足率、缺货率 |
| 提升调拨质量 | 调拨必须创造收益 | 调拨 ROI、调拨成功率 |
| 缩短逆向恢复 | 退货返仓尽快恢复可售 | 返仓可售率、48h 归因达标率 |

## 3. 输入字段

| 数据域 | 字段 | 来源 | 用途 |
|---|---|---|---|
| 仓储 | 仓、SKU、库位、可用、冻结、不良、预占 | WMS | 日清监控 |
| 库龄 | 入库日期、库龄段、90天+、180天+金额 | `dwt_inventory_health` | 清库治理 |
| 调拨 | 源仓、目标仓、运输成本、预计到达、调拨状态 | WMS/TMS | 调拨候选与验收 |
| 销售需求 | 日均销量、预测需求、断货风险 | `kp01` + `kp03` | 判断是否应调拨 |
| 逆向 | 退货原因、返仓状态、复检结果、报废 | `dwt_reverse_logistics` | 恢复可售 |

## 4. 指标与阈值

| 指标 | 公式/口径 | 阈值 | Owner |
|---|---|---|---|
| 仓储成本率 | `仓储费用/销售额` | 按品类校准 | 仓储经理 |
| 库龄>90天占比 | `90天+金额/总库存金额` | >10% 预警 | 库存计划 |
| 库龄>180天占比 | `180天+金额/总库存金额` | >5% 强预警 | 库存计划 |
| 库容利用率 | `实占方数/总方数` | <70% 或 >95% 提示 | 仓管 |
| 库存准确率 | `1-偏差率` | <98% 预警 | 仓管 |
| 调拨成功率 | `完成调拨/建议调拨` | <80% 提示 | 调拨经理 |
| 调拨ROI | `节省收益/调拨成本` | <1 复盘 | 调拨经理 |
| 返仓可售率 | `可再售件数/返仓件数` | <70% 预警 | 仓储主管 |
| 48h归因达标率 | `48h内归因/总退货` | <90% 预警 | 仓储主管 |

## 5. 执行 SOP

| 节奏 | 动作 | 输出 |
|---|---|---|
| 日度 | 监控可用、冻结、不良、预占和高风险 SKU 库存深度 | 日清异常表 |
| 日度 | 处理错发、返仓、复检超时 | 逆向处理清单 |
| 周度 | 召开 90天+、180天+ 清库会 | 清库动作表 |
| 周度 | 生成冗余仓到缺货仓调拨候选池 | 调拨候选单 |
| 月度 | 复盘周转、库龄、返仓恢复、仓储成本 | 仓储经营复盘 |

## 6. 调拨决策规则

| 条件 | 动作 |
|---|---|
| 目标仓库存覆盖 <25天 且源仓 >60天 | 自动进入调拨候选池 |
| 调拨 ROI <1 连续两周 | 暂停该策略并复盘 |
| 180天+库存占比 >5% | 强制处置或清库 |
| `z-score > 1.6` 且库存覆盖天数偏高 | 优先执行调拨+清库联动 |
| 返仓归因超 48h | 升级逆向专项 |

## 7. 阶段目标

| 时间 | 目标 |
|---|---|
| 60天 | 建立库存健康日看板和周清库会 |
| 90天 | 试点仓周转下降 10%，超龄库存显著下降 |
| 180天 | 仓储与调拨形成常态化协同机制，支撑周转进入 100 天以内 |

## 8. 节点接口

| 接口 | 内容 |
|---|---|
| 接收 `kp03-计划排产` | 到货与补货节奏 |
| 接收 `kp02-仓网规划` | 区域角色与优先级 |
| 输出给尾程 | 可售库存与订单履约能力 |
| 输出给逆向 | 返仓恢复率与报废原因反馈 |
