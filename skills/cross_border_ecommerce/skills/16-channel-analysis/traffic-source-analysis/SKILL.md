# 流量来源分析 (Traffic Source Analysis)

## 概述

流量来源分析是数据运营中的重要组成部分，涉及多种方法和指标，旨在优化用户获取和提升转化率。通过对不同流量来源的分析，企业可以了解用户从何处来，哪些渠道带来的用户质量更高。

## 核心功能

### 1. 流量来源分类

对流量进行系统性分类：
- **直接访问**: 用户直接输入网址或通过书签访问
- **外部链接**: 来自其他网站的引荐流量
- **搜索引擎**: 自然搜索(SEO)和付费搜索(SEM)
- **社交媒体**: 微博、微信、Facebook、Instagram等
- **广告投放**: 各类付费广告流量
- **邮件营销**: EDM带来的流量
- **应用程序**: 移动APP导流

### 2. 流量质量评估

多维度评估流量质量：
- Level 1: 访问时长、浏览页数、跳出率
- Level 2: 注册转化率
- Level 3: 购买转化率

### 3. 流量转化分析

使用漏斗模型分析各环节转化。

## 使用方法

### 输入要求

| 数据类型 | 必需字段 | 说明 |
|---------|---------|------|
| 访问日志 | session_id, source, medium | 会话和来源标识 |
| 用户数据 | user_id, is_new, device | 用户属性 |
| 行为数据 | pv, duration, bounce | 访问行为 |
| 转化数据 | conversions, revenue | 转化结果 |

### 分析流程

```
1. 流量分类
   ├── 识别流量来源
   ├── 归类到标准分类
   └── 标记流量属性

2. 质量评估
   ├── 计算基础指标(时长、页数、跳出率)
   ├── 计算转化指标(注册率、购买率)
   └── 综合评分

3. 转化分析
   ├── 构建转化漏斗
   ├── 识别流失环节
   └── 分析影响因素

4. 对比分析
   ├── 来源间对比
   ├── 时间趋势对比
   └── 同比/环比分析

5. 输出建议
   ├── 优质来源识别
   ├── 问题来源诊断
   └── 优化方向建议
```

## 核心公式

### 跳出率

```
跳出率 = 单页访问会话数 / 总会话数 × 100%
```

### 平均访问时长

```
平均时长 = 总访问时长 / 会话数
```

### 转化率

```
转化率 = 完成目标用户数 / 总访问用户数 × 100%
```

### 流量质量指数

```
TQI = (1-跳出率)×0.3 + min(时长/180, 1)×0.3 + 转化率×10×0.4
```

## 示例

### 示例1: 母婴电商流量来源分析

**场景**: 某母婴跨境电商月度流量分析

**数据概览**:
| 来源 | UV | PV | 跳出率 | 时长 | 转化率 |
|-----|-----|-----|--------|-----|--------|
| SEO | 50,000 | 200,000 | 32% | 195s | 4.2% |
| SEM | 30,000 | 105,000 | 38% | 165s | 5.8% |
| Facebook | 25,000 | 75,000 | 45% | 135s | 3.1% |
| Instagram | 15,000 | 42,000 | 52% | 105s | 2.4% |
| EDM | 8,000 | 32,000 | 25% | 225s | 7.2% |
| 直接访问 | 20,000 | 80,000 | 28% | 210s | 5.5% |

**分析结论**:
1. SEO流量质量最高，建议加强内容营销
2. EDM转化率最高，老客复购价值大
3. Instagram跳出率高，需优化落地页

**优化建议**:
- SEO: 增加长尾关键词覆盖
- 社媒: 优化素材与落地页一致性
- EDM: 提升发送频率和个性化

### 示例2: 流量异常诊断

**场景**: 本周流量突然下降30%

**诊断步骤**:
1. 分来源检查: SEM下降50%，其他正常
2. 检查SEM细分: 品牌词正常，通用词下降
3. 定位原因: 竞品加大投放，竞价排名下降

**解决方案**:
- 短期: 增加核心词预算
- 中期: 优化质量得分
- 长期: 加强SEO建设

## 注意事项

### 数据层面
1. **UTM参数**: 确保所有推广链接正确标记UTM参数
2. **跨设备**: 注意跨设备用户识别问题
3. **数据延迟**: 实时数据与最终数据可能有差异

### 分析层面
1. **区分自然与付费**: 同一渠道需区分自然和付费流量
2. **作弊流量**: 警惕刷量流量，关注核心转化指标
3. **归因周期**: 设定合理的转化归因窗口

### 业务层面
1. **新客vs老客**: 区分新流量和回访流量
2. **品牌词vs通用词**: 分析搜索流量时区分词类
3. **移动vs桌面**: 不同设备流量特点不同
