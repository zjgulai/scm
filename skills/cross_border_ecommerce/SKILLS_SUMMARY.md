**功能**: 检测和处理数据质量问题

**触发词**: 数据清洗、数据预处理、缺失值、异常值、去重

**输入**:
```
原始数据文件 (CSV/Excel)

```

**输出**:
```markdown
# 数据清洗报告

## 数据质量评估
| 指标 | 数值 | 状态 |
|------|------|------|
| 总记录数 | 10,000 | - |
| 缺失值比例 | 2.3% | ⚠️ 需处理 |
| 异常值数量 | 45 | ⚠️ 需处理 |
| 重复记录 | 12 | ⚠️ 需处理 |

## 清洗操作
1. 缺失值处理: 金额字段用中位数填充
2. 异常值处理: 删除金额>3σ的记录
3. 重复处理: 删除12条重复记录

## 清洗后数据
- 有效记录: 9,943条
- 数据完整率: 99.4%
```

```markdown
# 销售额变动归因分析

## 变动概览

- 本期销售额: ¥594,000
- 变动: +¥94,000 (+18.8%)

## 因素贡献度分解

| 因素 | 基期值 | 本期值 | 变动 | 贡献金额 | 贡献占比 |
|------|--------|--------|------|----------|----------|
| 访客数 | 100,000 | 120,000 | +20% | +¥100,000 | +106.4% |
| 转化率 | 5.0% | 4.5% | -10% | -¥50,000 | -53.2% |
| 客单价 | ¥100 | ¥110 | +10% | +¥50,000 | +53.2% |
| 交互效应 | - | - | - | -¥6,000 | -6.4% |
| **合计** | - | - | - | **+¥94,000** | **100%** |

## 关键发现
1. 访客数增长是主要驱动力 (+106.4%)
2. 转化率下降拖累了50%以上的增长
3. 建议优先优化转化率

## 优化建议
- 分析转化率下降原因(页面体验?流量质量?)
- 保持访客增长势头
- 客单价提升效果良好，可继续优化
```

```

基期数据:
- 销售额: 250万

- ROI: 1.5

本期数据:
- 销售额: 280万
- 成本: 140万
- ROI: 1.0
```

**输出**:
```markdown
# 复杂指标贡献度分析报告

## 指标关系
ROI = (销售额 - 成本) / 成本

## 变动概览
- 基期ROI: 1.50
- 本期ROI: 1.00
- 总变动: -0.50

## 静态贡献度（敏感度）
| 因素 | 偏导数 | 弹性系数 | 影响方向 |
|------|--------|---------|---------|
| 销售额 | 0.01 | 1.67 | 正向 |
| 成本 | -0.025 | -1.67 | 负向 |

## 波动贡献度
| 因素 | 变动值 | 波动贡献 | 贡献占比 |
|------|--------|---------|---------|
| 销售额 | +30万 | +0.30 | -60% |
| 成本 | +40万 | -1.00 | +200% |
| 交互效应 | - | +0.20 | -40% |
| **合计** | - | **-0.50** | **100%** |

## 关键结论
1. 成本上升是ROI下降的主因 (贡献200%)
2. 销售额增长部分抵消了负面影响
3. 交互效应显著(-40%)，需关注销售额-成本联动

## 优化建议
- 优先控制成本增长
- 提升销售额增速，超过成本增速
- 关注成本效率(ROI = 利润率/成本率)
```

## 四、Skills使用最佳实践

### 4.1 分析流程模板

```
完整的业务分析流程:

1. 数据准备
   └── data-cleaning: 清洗数据，确保质量

2. 现状了解
   └── data-analyst: 获取数据概览
   └── yoy-mom-analysis: 同比环比趋势

3. 问题定位
   └── contribution-calculation: 归因分析
   └── complex-contribution: 复杂归因
   └── structural-analysis: 结构拆解

4. 深度分析
   └── 按领域选择: margin-attribution / channel-effect-analysis 等

5. 输出报告
   └── dashboard-design: 看板设计
   └── data-reporting: 报告撰写
```

### 4.2 常见问题解决路径

**问题1: 销售额下降了，怎么分析？**
```
销售额下降分析路径:

1. yoy-mom-analysis → 确认下降幅度和时间
2. contribution-calculation → 拆解访客×转化×客单价
3. traffic-source-analysis → 分析流量来源变化
4. channel-effect-analysis → 评估各渠道表现
5. anomaly-detection → 检测是否有异常
```

**问题2: 毛利率下滑了，怎么定位？**
```
毛利率下滑分析路径:

1. margin-attribution → 结构vs自身变动
2. binning-contribution → 按客单价/客品数分箱
3. cost-structure-analysis → 成本结构分析
4. complex-contribution → 复杂公式归因
```

**问题3: 广告效果不好，怎么优化？**
```
广告优化分析路径:

1. ad-performance-analysis → 广告效果评估
2. channel-effect-analysis → 渠道对比
3. traffic-source-analysis → 流量质量
4. ad-attribution → 归因和预算分配
```
