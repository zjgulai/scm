## 月度经营复盘工作流

### 业务场景

运营经理需要每月对业务进行全面复盘，了解业绩表现、识别问题、制定改进计划。

### 参与Skills

| Skill | 用途 | 输出 |
|-------|------|------|
| `cbec-data-cleaning` | 清洗多源数据 | 标准化数据集 |
| `cbec-yoy-mom-analysis` | 趋势分析 | 增长率报告 |
| `cbec-contribution-calculation` | 归因分析 | 因素贡献度 |
| `cbec-profitability-matrix` | 产品分析 | 产品分层 |

### 详细步骤

**步骤1: 数据准备**

收集以下数据文件:
- `sales_2024_03.csv` - 3月销售数据
- `ad_spend_2024_03.csv` - 3月广告支出
- `cost_2024_03.csv` - 3月成本数据
- `products.csv` - 产品信息

**步骤2: 数据清洗**

对话示例:
```
用户: 请帮我清洗这个月的销售数据 [上传sales_2024_03.csv]

Claude: [加载cbec-data-cleaning]
"我将为您清洗销售数据..."

分析:
- 原始记录: 15,000条
- 缺失值: 200条 (已填充)
- 异常值: 50条 (已标记)
- 重复记录: 30条 (已删除)

输出: sales_2024_03_cleaned.csv
```

```

└─────────┴─────────┴─────────┴────────┴─────────┘

关键发现:
1. YoY增长25%，表现优异

```

**步骤4: 归因分析**

对话示例:
```
用户: 分析销售额增长的驱动因素

Claude: [加载cbec-contribution-calculation]

销售额 = 访客数 × 转化率 × 客单价

因素贡献分析:
┌───────────┬────────┬────────┬─────────┬──────────┐
│ 因素      │ 2月    │ 3月    │ 变动    │ 贡献度   │
├───────────┼────────┼────────┼─────────┼──────────┤
│ 访客数    │ 50K    │ 58K    │ +16%    │ 60%      │
│ 转化率    │ 5.6%   │ 5.2%   │ -7%     │ -20%     │
│ 客单价    │ $48    │ $50    │ +4%     │ 20%      │
├───────────┼────────┼────────┼─────────┼──────────┤
│ 总变动    │        │        │ +$15K   │ 100%     │
└───────────┴────────┴────────┴─────────┴──────────┘

结论:
- 访客数增长是主要驱动因素(60%贡献)
- 转化率下降抵消了部分增长(-20%)
- 客单价提升贡献正向(20%)

建议:
- 继续加大流量获取
- 优化页面转化，提升转化率
```

## 新品上市定价工作流

### 业务场景

产品经理准备推出新产品，需要确定最优售价。

### 参与Skills

| Skill | 用途 | 输出 |
|-------|------|------|
| `cbec-cost-structure-analysis` | 成本分析 | 成本结构 |
| `cbec-price-elasticity` | 弹性分析 | 价格敏感度 |
| `cbec-unit-economics` | 盈利分析 | 利润预测 |

### 工作流程图

```
输入:
├── 产品成本数据
├── 竞品价格数据
└── 历史类似产品销售数据

步骤1: 成本分析
[cbec-cost-structure-analysis]
├── 计算产品成本
├── 计算物流成本
├── 计算平台费用
└── 确定成本基准

步骤2: 价格弹性分析
[cbec-price-elasticity]
├── 分析历史价格-销量关系
├── 计算弹性系数
└── 确定定价区间

步骤3: 单位经济分析
[cbec-unit-economics]
├── 计算不同价格下的利润
├── 预测销量
└── 评估盈利可行性

步骤4: 定价决策
├── 综合成本、竞争、盈利目标
└── 确定最终售价

输出:
├── 定价分析报告
├── 利润预测表
└── 定价建议
```

### 详细步骤

**步骤1: 成本分析**

输入数据:
```csv
product_name,procurement_cost,packaging_cost,headway_fee,fulfillment_fee,platform_fee_rate
新品耳机,80,5,15,12,0.15
```

对话示例:
```
用户: 分析新耳机的成本结构

Claude: [加载cbec-cost-structure-analysis]

成本结构分析:
┌──────────────┬────────┬──────────┐
│ 成本项       │ 金额   │ 占比     │
├──────────────┼────────┼──────────┤
│ 采购成本     │ $80    │ 66.7%    │
│ 包装成本     │ $5     │ 4.2%     │
│ 头程运费     │ $15    │ 12.5%    │
│ 尾程配送     │ $12    │ 10.0%    │
│ 平台费用(15%)│ ?      │ 15.0%    │
├──────────────┼────────┼──────────┤
│ 总成本(不含平台费)│ $112 │ 93.3%  │
└──────────────┴────────┴──────────┘

成本基准: $112 + 15%平台费
```

**步骤2: 价格弹性分析**

输入历史数据:
```csv
price,sales_quantity
100,500
110,450
120,380
130,300
```

对话示例:
```
用户: 分析类似产品的价格弹性

Claude: [加载cbec-price-elasticity]

价格弹性分析:
- 弹性系数: -1.5 (弹性较大)
- 价格敏感度: 高
- 最优价格区间: $110-$125

价格-销量关系:
价格$110: 预计销量450件，收入$49.5K
价格$120: 预计销量380件，收入$45.6K
价格$125: 预计销量340件，收入$42.5K

结论: 价格敏感度高，不宜定价过高
```

**步骤3: 单位经济分析**

对话示例:
```
用户: 分析不同价格下的盈利情况

Claude: [加载cbec-unit-economics]

盈利分析:
┌───────┬────────┬────────┬────────┬────────┬────────┐
│ 价格  │ 销量   │ 收入   │ 成本   │ 利润   │ 利润率 │
├───────┼────────┼────────┼────────┼────────┼────────┤
│ $110  │ 450    │ $49.5K │ $46.2K │ $3.3K  │ 6.7%   │
│ $120  │ 380    │ $45.6K │ $39.0K │ $6.6K  │ 14.5%  │
│ $125  │ 340    │ $42.5K │ $34.9K │ $7.6K  │ 17.9%  │
│ $130  │ 300    │ $39.0K │ $30.8K │ $8.2K  │ 21.0%  │
└───────┴────────┴────────┴────────┴────────┴────────┘

注: 成本 = 固定成本($112×销量) + 平台费(15%×收入)

推荐价格: $125
- 利润率: 17.9% (健康)
- 销量: 340件 (合理)
- 利润: $7.6K (最大化)
```

**步骤4: 定价决策**

综合建议:
```
定价建议报告

推荐售价: $124.99

理由:
1. 成本覆盖: 完全覆盖所有成本
2. 利润率: 17.8%，符合公司标准(>15%)
3. 市场竞争力: 处于竞品价格带中游
4. 价格弹性: 在弹性可接受范围内

风险提示:
- 如竞品降价，需跟进调整
- 建议首月测试$119.99促销价

预期效果:
- 月销量: 350件
- 月收入: $43.7K
- 月利润: $7.8K
```

## 年度财务规划工作流

### 业务场景

财务总监需要制定下一年度的财务预算和规划。

### 参与Skills

| Skill | 用途 | 输出 |
|-------|------|------|
| `cbec-dupont-analysis` | ROE分解 | 财务诊断 |
| `cbec-yoy-mom-analysis` | 趋势分析 | 历史趋势 |
| `cbec-demand-forecasting` | 销售预测 | 收入预测 |

### 工作流程

**步骤1: 财务诊断**

对话示例:
```
用户: 分析公司当前财务状况

Claude: [加载cbec-dupont-analysis]

杜邦分析结果:
当前ROE: 28%

分解:
- 净利润率: 10% (行业平均12%)
- 资产周转率: 2.0次 (行业平均1.8次)
- 权益乘数: 1.4 (行业平均1.5)

分析:
- 优势: 资产周转效率高
- 劣势: 净利润率低于行业
- 建议: 优化成本结构，提升利润率
```

**步骤2: 趋势分析**

分析过去3年财务趋势，识别增长模式。

| `cbec-association-analysis` | 关联分析 | 连带购买 |

### 工作流程

**步骤1: 销售归因**

对话示例:
```
用户: 分析黑五促销带来的增量销售

Claude: [加载cbec-variance-attribution]

促销效果分析:
- 促销期销售: $500K
- 预期销售(无促销): $350K
- 增量销售: $150K (+43%)

归因:
- 促销折扣贡献: $80K
- 广告增量贡献: $50K
- 自然增长: $20K
```

**步骤2: 成本分析**

计算促销总成本:
- 折扣成本: $50K
- 广告成本: $30K
- 物流成本增加: $10K
- 总成本: $90K

**步骤3: ROI计算**

```
促销ROI = 增量利润 / 促销成本
        = ($150K × 20%毛利率) / $90K
        = $30K / $90K
        = 33%

结论: ROI偏低，需优化促销策略
```

**步骤4: 关联分析**

分析促销期间的购物篮，发现连带购买模式。
