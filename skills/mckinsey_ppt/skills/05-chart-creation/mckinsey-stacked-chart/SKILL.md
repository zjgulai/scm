# 麦肯锡堆叠柱状图制作 Skill

## 概述

堆叠柱状图(Stacked Bar Chart)是麦肯锡展示结构分析和年龄/时间维度对比的首选图表，特别适合展示"部分与整体"的关系。

## 设计规范

### 1. 100%堆叠原则

```
每个柱子高度 = 100%
确保视觉对比公平

示例:
年龄段A: 品牌68% + 设计20% + 其他12% = 100%
年龄段B: 品牌50% + 设计30% + 其他20% = 100%
```

### 2. 排序规则

**类别排序**:
```
- 最重要的类别放在底部
- 保持各柱类别顺序一致
- 使用颜色区分类别
```

**柱子排序**:
```
- 按逻辑顺序(如年龄从小到大)
- 或按某个关键指标排序
```

### 3. 颜色配置

```css
/* 麦肯锡堆叠图配色 */
.brand { color: #051C2C; }      /* 品牌 - 深蓝 */
.design { color: #1E3A5F; }     /* 设计 - 中蓝 */
.process { color: #3A5F8A; }    /* 生产 - 浅蓝 */
.material { color: #5A7F9F; }   /* 材质 - 淡蓝 */
.price { color: #999999; }      /* 价格 - 灰色 */
```

### 4. 数据标注

```
方式1: 柱内标注(空间足够时)
┌──────────────┐
│     68%      │ ← 设计占比
│   ████████   │
│     32%      │ ← 品牌占比
│   ████████   │
└──────────────┘

方式2: 柱外标注(空间不足时)
┌──────────────┐ 68% 设计
│   ████████   │

方式3: 仅标注关键数据
┌──────────────┐
│   ████████   │ 68%
│   ████████   │
└──────────────┘
```

## 代码示例

### Python (Matplotlib)

```python
import matplotlib.pyplot as plt
import numpy as np

# 数据
age_groups = ['1990年后', '1965-1970年', '1945-1950年']
brand = [32, 50, 94]
design = [68, 50, 6]

# 颜色
colors = ['#051C2C', '#3A5F8A']

# 创建图表
fig, ax = plt.subplots(figsize=(10, 6))

# 绘制堆叠柱状图
x = np.arange(len(age_groups))
width = 0.6

ax.bar(x, brand, width, label='品牌', color=colors[0])
ax.bar(x, design, width, bottom=brand, label='设计/风格', color=colors[1])

# 添加标签
ax.set_ylabel('占比 (%)')
ax.set_xticks(x)
ax.set_xticklabels(age_groups)
ax.legend()

plt.tight_layout()
plt.show()
```

## 何时不要使用

- 需求仅是简单文案润色或排版修饰，不涉及本 Skill 对应的方法或图表决策。
- 用户问题与该 Skill 的核心场景不匹配，存在更直接的专项 Skill 可用。
- 仅需一次性小修改且不需要此 Skill 的方法论框架。

## 快速使用清单

- 明确页面目标或分析问题（先结论后图表）。
- 确认输入数据字段、时间口径、分组维度。
- 选择图表类型或方法模板并执行。
- 产出标题、关键结论、图表与数据来源。
- 用本 Skill 的质量检查项完成最终自检。

## 常见错误

- 图表类型与业务问题不匹配，导致结论表达失焦。
- 标题不是结论句，只写成章节名。
- 过度堆叠信息，单页承载超过可读上限。
- 缺失数据口径和来源说明，影响复用与可信度。
