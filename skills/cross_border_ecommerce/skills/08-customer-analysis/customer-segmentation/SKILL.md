# 母婴跨境电商客户分群分析 Skill

## 概述

本Skill基于**RFM+L模型**（Recency最近购买、Frequency购买频率、Monetary消费金额、**Lifecycle宝宝生命周期**）实现母婴跨境电商客户分群分析。通过科学的客户分层，帮助运营团队识别高价值客户、制定差异化运营策略，提升客户终身价值（CLV）。

### 母婴行业特性

母婴客户具有独特的生命周期特征:
- **决策周期长**: 从孕期开始研究，多方比较后购买
- **安全性优先**: 高度关注产品成分、认证、口碑
- **社群影响大**: 妈妈群、KOL种草影响显著
- **生命周期明确**: 孕期→新生儿→婴幼儿→学龄前
- **品类随宝宝成长迁移**: 需求随宝宝月龄变化

本Skill充分考虑这些特性，提供母婴专属的分群维度和运营策略。

## 核心功能

### 1. RFM+L模型分析
- **Recency (R)**: 最近一次购买距今天数
- **Frequency (F)**: 分析周期内购买次数
- **Monetary (M)**: 分析周期内消费金额
- **Lifecycle (L)**: **母婴专属** - 宝宝生命周期阶段(孕期/新生儿/婴幼儿/学龄前)
- **RFM+L评分**: R/F/M各1-5分，L为阶段标识，组合成客户群体

### 2. 母婴专属生命周期分析

#### 宝宝生命周期阶段
| 阶段 | 时间范围 | 主要需求品类 | 典型产品 |
|------|---------|-------------|---------|
| 孕期客户 | 孕期0-9个月 | 孕期营养、待产包、孕妇装 | 叶酸、孕妇维生素、待产包 |
| 新生儿期 | 0-6个月 | 奶粉、纸尿裤、婴儿床、奶瓶 | 配方奶粉、NB/S号纸尿裤 |
| 婴幼儿期 | 6-24个月 | 辅食、玩具、早教、童装 | 辅食泥、爬行垫、早教玩具 |
| 学龄前期 | 2-6岁 | 教育玩具、童装、图书、安全座椅 | 乐高、绘本、平衡车 |

#### 品类迁移路径
母婴客户的需求会随宝宝成长自然迁移:
```
孕期 → 新生儿期 → 婴幼儿期 → 学龄前期
 ↓        ↓          ↓          ↓
孕期营养  奶粉+纸尿裤  辅食+玩具   教育玩具
待产包    婴儿床      早教产品    童装
孕妇装    奶瓶        爬行垫      图书
```

**分析要点**:
- 追踪客户品类迁移时机
- 预测下一阶段需求
- 识别迁移失败客户(流失风险)

### 3. 客户分群
基于RFM评分将客户分为8个群体，并叠加生命周期阶段：

| 群体 | RFM特征 | 特点 | 运营策略(母婴专属) |
|------|---------|------|---------|
| 重要价值客户 | R↑ F↑ M↑ | 最近购买、频率高、金额大 | VIP服务、订阅制优惠、新品优先试用 |
| 重要发展客户 | R↑ F↓ M↑ | 最近购买、金额大、频次低 | 提升复购、订阅转化、月龄产品推荐 |
| 重要保持客户 | R↓ F↑ M↑ | 频率高、金额大、但最近未购 | 唤醒激活、品类迁移提醒、流失预警 |
| 重要挽留客户 | R↓ F↓ M↑ | 金额大、但频次低且近期未购 | 重点挽回、调研原因、专属回归优惠 |
| 一般价值客户 | R↑ F↑ M↓ | 最近购买、频率高、金额低 | 客单价提升、关联推荐、品类扩展 |
| 一般发展客户 | R↑ F↓ M↓ | 最近购买、但频次金额都低 | 新客培育、育儿知识推送、信任建立 |
| 一般保持客户 | R↓ F↑ M↓ | 频率高、但近期未购且金额低 | 常规激活、促销触达、社群互动 |
| 低价值客户 | R↓ F↓ M↓ | 各项指标都低 | 低成本维护、自动化营销、筛选潜力 |

### 4. 母婴专属分群维度

#### 4.1 订阅制客户分群
| 订阅类型 | 特征 | 占比目标 | 价值特点 |
|---------|------|---------|---------|
| 高频订阅客户 | 纸尿裤/奶粉月度订阅 | 15-20% | 高复购、高LTV、低流失 |
| 中频订阅客户 | 季度订阅 | 10-15% | 稳定复购、可升级空间 |
| 非订阅高价值 | 高RFM但未订阅 | 20-30% | **订阅转化重点目标** |
| 订阅流失客户 | 曾订阅但取消 | 5-10% | 调研原因、挽回策略 |

#### 4.2 宝宝月龄分群
| 月龄段 | 客户特征 | 需求特点 | 关键指标 |
|-------|---------|---------|---------|
| 孕期(准妈妈) | 决策周期长、安全关注高 | 待产包、孕期营养 | 首单转化率、客单价 |
| 0-6个月 | 高频刚需、品牌忠诚建立 | 奶粉、纸尿裤 | 复购率、订阅转化率 |
| 7-12个月 | 品类扩展、辅食引入 | 辅食、玩具、日用品 | 品类数、交叉购买率 |
| 13-24个月 | 品类迁移、需求变化 | 早教、童装、玩具 | 迁移成功率、留存率 |
| 25个月+ | 稳定期或流失风险 | 教育产品、大龄童装 | 留存率、LTV |

#### 4.3 二胎/多胎客户分群
| 类型 | 特征 | 价值 | 策略 |
|------|------|------|------|
| 二胎孕期 | 已有购买经验、决策快 | 高转化、高客单 | 快速推荐、优惠激励 |
| 二胎新生儿 | 复购刚需品、可能升级 | 高复购、品牌忠诚 | 订阅转化、新品推荐 |
| 多胎同时 | 多宝宝需求叠加 | 超高价值 | VIP服务、批量优惠 |

### 5. 传统客户生命周期分析
- **新手期**: 首次购买后30天内
- **成长期**: 购买2-3次，逐步建立信任
- **成熟期**: 购买4次以上，形成稳定习惯
- **衰退期**: 超过90天未购买
- **流失期**: 超过180天未购买

### 6. CLV计算(母婴增强版)
客户终身价值 = 平均客单价 × 年购买频率 × 平均生命周期（年）

**母婴专属调整**:
- **生命周期调整**: 考虑宝宝成长周期，典型母婴客户生命周期2-4年
- **品类迁移价值**: 计算品类迁移带来的增量价值
- **订阅价值加成**: 订阅客户LTV通常比非订阅客户高30-50%
- **二胎价值**: 二胎客户LTV是头胎客户的1.5-2倍

**母婴CLV公式**:
```
母婴CLV = (基础CLV × 品类迁移系数) + 订阅价值加成 + 二胎潜在价值

其中:
- 品类迁移系数 = 1 + (成功迁移次数 × 0.15)
- 订阅价值加成 = 订阅月数 × 月订阅额 × 0.2
- 二胎潜在价值 = (客户年龄 - 首胎年龄 < 35) × 基础CLV × 0.3
```

## 使用方法

### 分析流程

```
步骤1: 数据准备
- 加载客户交易数据
- 数据清洗与验证
- 设定分析周期（默认365天）
- 【母婴专属】提取宝宝月龄/预产期信息

步骤2: 计算RFM+L值
- R: (分析截止日期 - 最后购买日期) 的天数
- F: 分析周期内订单数量
- M: 分析周期内消费总额
- L: 【母婴专属】计算宝宝生命周期阶段
  * 如果有baby_due_date且未到预产期 → 孕期
  * 如果有baby_birth_date → 计算月龄，映射到阶段
  * 如果无宝宝信息 → 根据购买品类推断阶段

步骤3: RFM+L评分
- 将R/F/M分别分为5个等级（1-5分）
- R: 分数越高表示越近期购买
- F/M: 分数越高表示频次/金额越高
- L: 【母婴专属】标识生命周期阶段

步骤4: 客户分群
- 基于RFM评分组合划分8个群体
- 叠加生命周期阶段标签
- 【母婴专属】识别订阅客户、二胎客户
- 计算各群体人数、占比、贡献

步骤5: 生命周期分析
- 判断客户所处生命周期阶段
- 计算阶段分布与迁移
- 【母婴专属】分析品类迁移路径
- 【母婴专属】预测下一阶段需求

步骤6: 策略建议
- 为每个群体生成运营策略建议
- 【母婴专属】提供母婴专属运营建议
- 输出可执行的营销方案
```

## RFM评分标准

### Recency评分 (R)

| 分数 | 条件 | 说明 |
|------|------|------|
| 5 | ≤30天 | 非常活跃 |
| 4 | 31-60天 | 活跃 |
| 3 | 61-90天 | 正常 |
| 2 | 91-180天 | 不活跃 |
| 1 | >180天 | 沉睡 |

### Frequency评分 (F)

| 分数 | 条件 | 说明 |
|------|------|------|
| 5 | ≥12次/年 | 高频 |
| 4 | 6-11次/年 | 较高频 |
| 3 | 4-5次/年 | 中频 |
| 2 | 2-3次/年 | 低频 |
| 1 | 1次/年 | 单次购买 |

### Monetary评分 (M)

使用分位数划分（可根据业务调整）：
| 分数 | 条件 | 说明 |
|------|------|------|
| 5 | >第80百分位 | 高消费 |
| 4 | 第60-80百分位 | 较高消费 |
| 3 | 第40-60百分位 | 中等消费 |
| 2 | 第20-40百分位 | 较低消费 |
| 1 | <第20百分位 | 低消费 |

## 核心公式

### RFM+L评分
```
R_score = f(距今天数) → 1-5分
F_score = g(购买次数) → 1-5分
M_score = h(消费金额) → 1-5分
L_stage = 宝宝生命周期阶段(孕期/新生儿/婴幼儿/学龄前)

RFM+L组合 = [R_score, F_score, M_score, L_stage]
```

### 宝宝月龄计算
```
如果 baby_birth_date 存在:
    宝宝月龄 = (当前日期 - baby_birth_date) / 30

如果 baby_due_date 存在且未到预产期:
    生命周期阶段 = "孕期"
    孕期阶段 = (预产期 - 当前日期) / 30 (剩余月份)

生命周期阶段映射:
- 孕期(0-9个月): 孕期客户
- 0-6个月: 新生儿期
- 7-24个月: 婴幼儿期
- 25个月+: 学龄前期
```

### 客户分群规则(含母婴扩展)
```
# 基础RFM分群
if R_score >= 4 and F_score >= 4 and M_score >= 4:
    基础群体 = "重要价值客户"
elif R_score >= 4 and F_score < 4 and M_score >= 4:
    基础群体 = "重要发展客户"
elif R_score < 4 and F_score >= 4 and M_score >= 4:
    基础群体 = "重要保持客户"
elif R_score < 4 and F_score < 4 and M_score >= 4:
    基础群体 = "重要挽留客户"
elif R_score >= 4 and F_score >= 4 and M_score < 4:
    基础群体 = "一般价值客户"
elif R_score >= 4 and F_score < 4 and M_score < 4:
    基础群体 = "一般发展客户"
elif R_score < 4 and F_score >= 4 and M_score < 4:
    基础群体 = "一般保持客户"
else:
    基础群体 = "低价值客户"

# 母婴专属标签
if subscription_status == True:
    客户标签 += "订阅客户"

if has_second_child == True:
    客户标签 += "二胎客户"

if L_stage in ["品类迁移期"]:
    客户标签 += "迁移风险"

最终群体 = 基础群体 + " " + 客户标签
```

### CLV计算(母婴增强版)
```
# 基础CLV
平均客单价 = 总消费金额 / 总订单数
年购买频率 = 总订单数 / 活跃客户数 / 分析年数
平均生命周期 = 平均客户活跃天数 / 365
基础CLV = 平均客单价 × 年购买频率 × 平均生命周期

# 母婴CLV调整
品类迁移系数 = 1 + (成功迁移次数 × 0.15)
订阅价值加成 = 订阅月数 × 月订阅额 × 0.2 (如果订阅)
二胎潜在价值 = (客户年龄 - 首胎年龄 < 35) × 基础CLV × 0.3 (如果有二胎潜力)

母婴CLV = (基础CLV × 品类迁移系数) + 订阅价值加成 + 二胎潜在价值
```

### 品类迁移成功率
```
品类迁移成功率 = 成功迁移客户数 / 应迁移客户总数

其中:
应迁移客户 = 宝宝月龄超过当前阶段 + 6个月的客户
成功迁移 = 在应迁移后3个月内购买新品类的客户
```

### 订阅转化率
```
订阅转化率 = 新增订阅客户数 / 目标客户数

目标客户筛选条件:
- R_score >= 3 (近期有购买)
- 购买品类 in [纸尿裤, 奶粉, 辅食] (高频刚需)
- 宝宝月龄 in [0-12个月] (订阅需求强)
- 当前非订阅客户
```
