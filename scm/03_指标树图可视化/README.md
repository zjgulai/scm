# Momcozy 供应链 KPI 指标树图可视化

本目录包含 Momcozy 品牌出海供应链 **综合满意度 KPI 体系** 的多种可视化格式。

## 📐 计算公式（原著引用）

### 综合供应链满意度（融合设计公式）
```
综合供应链满意度 = 成本满意度得分 × 50% + 履约满意度得分 × 50%

成本满意度得分 = 100 - (实际全链路成本率 / 18% × 100)
（成本率越低得分越高，封顶100分，保底0分）

履约满意度得分 = 实际综合履约满意度指数得分
（基于时效40% + 质量40% + 成本效率20%的加权）
```

### 原著公式来源

#### 成本维度公式（第三章第五节「进-存-销成本结构」）
| 指标 | 计算公式 | 原著来源 |
|------|---------|---------|
| 全链路供应链成本率 | 供应链总成本 ÷ GMV × 100% | 第三章第五节 |
| 头程运输成本率 | 头程运费 ÷ 进货货值 × 100% | 第三章第五节「进」的成本 |
| 仓储成本率 | 月度仓储费 ÷ 月均库存货值 × 100% | 第三章第五节「存」的成本 |
| 资金占用成本率 | 平均库存金额 × 资金年化成本率 ÷ 年销售额 × 100% | 第三章第五节「存」的成本（资金时间价值） |
| 库存损耗率 | (盘亏+报损+过期) ÷ 平均库存金额 × 100% | 第三章第五节「存」的成本 |
| 尾程配送成本率 | 尾程配送费 ÷ 销售额 × 100% | 第三章第五节「销」的成本 |
| 订单履约成本率 | (拣货+打包+贴标+出库)成本 ÷ 订单数 | 第二章第四节「物流成本KPI」 |

#### 履约维度公式（第二章第四节）
| 指标 | 计算公式 | 原著来源 |
|------|---------|---------|
| 综合履约满意度指数 | 时效×40% + 质量×40% + 成本效率×20% | 第二章第四节「成本、体验KPI」 |
| 整体履约时效 | 订单确认到签收的平均小时数 | 第二章第四节「时效KPI」 |
| 承诺时效达成率 | 实际时效≤承诺时效的订单数 ÷ 总订单数 × 100% | 第二章第四节「时效KPI」 |
| 发货及时率 | 24h内发货的订单数 ÷ 总订单数 × 100% | 第二章第四节「B2C配送KPI」 |
| 履约准确率 | [1-(错发+漏发+多发)÷总订单] × 100% | 第二章第四节「履约质量KPI」 |
| 破损丢失率 | (破损订单+丢失订单) ÷ 总订单数 × 100% | 第二章第四节「破损丢失投诉率」 |
| 现货满足率 | 有库存可发的SKU数 ÷ 总SKU数 × 100% | 第二章第二节「满足率KPI」 |
| 一次解决率 | 首次联系即解决的客诉数 ÷ 总客诉数 × 100% | 服务质量维度 |

#### 执行层公式（第二章 + 第三章）
| 指标 | 计算公式 | 原著来源 |
|------|---------|---------|
| 海运单位成本 | 海运总费用 ÷ 总立方数(CBM) | 第三章第五节 |
| 库存周转天数 | 平均库存金额 ÷ (销售成本÷365) | 第二章第二节「货品效率KPI」 |
| 库龄结构健康度 | 库龄<90天库存金额 ÷ 总库存金额 × 100% | 第三章第六节「健康库存」 |
| 盘点准确率 | [1-\|系统库存-实物库存\|÷总库存] × 100% | 第二章第四节「仓储管理KPI」 |
| 拣货准确率 | [1-拣货错误次数÷总拣货单数] × 100% | 第二章第四节 |
| 订单满足率 | 完全满足的订单数 ÷ 总订单数 × 100% | 第二章第二节「满足率KPI」 |

### 公式使用说明
- **所有公式均引用原著**，已在各指标卡片中标注具体章节
- HTML 交互版中点击任意指标可查看完整公式和原著来源
- Markdown 版本中每个指标下方都有详细的计算公式说明

### 评分等级
| 等级 | 得分区间 | 状态 | 行动建议 |
|------|---------|------|---------|
| A | 90-100 | 🟢 卓越 | 保持并寻求优化空间 |
| B | 80-89 | 🟡 良好 | 维持现状，关注波动 |
| C | 70-79 | 🟠 达标 | 需针对性改进 |
| D | 60-69 | 🔴 预警 | 紧急制定改进计划 |
| E | <60 | ⚫ 危险 | 高层介入，全面复盘 |

## 🗂️ 指标层级说明

| 层级 | 名称 | 说明 | 示例 |
|------|------|------|------|
| L0 | 终极目标 | 综合供应链满意度 | ≥80分 |
| L1 | 维度层 | 两大维度各占50% | 成本满意度 / 履约满意度 |
| L2 | 战略层 | 公司级核心指标 | 全链路成本率 ≤18% |
| L3 | 模块层 | 业务模块指标 | 进货/存货/销货成本 |
| L4 | 指标层 | 具体监控指标 | 头程运输成本率 ≤8% |
| L5 | 渠道层 | 按渠道拆分 | FBA/FBM/DTC |
| L6 | 执行层 | 操作级指标 | 海运$/CBM ≤$150 |

## 📥 XMind 导入详细步骤

### 从 Markdown 导入

1. **打开 XMind**
   - 启动 XMind Zen 或 XMind 2024

2. **选择导入**
   ```
   文件 → 导入 → Markdown
   ```

3. **选择文件**
   - 浏览到 `xmind/Momcozy_融合指标树.md`
   - 点击「打开」

4. **调整样式**（推荐）
   - 中心主题：图标 🏭，填充色 #E74C3C
   - 成本维度：分支颜色 #3498DB（蓝色）
   - 履约维度：分支颜色 #2ECC71（绿色）
   - 添加优先级标记到关键指标

5. **保存为 XMind 格式**
   ```
   文件 → 另存为 → 选择.xmind格式
   ```

## ❓ 常见问题

**Q: XMind 文件打不开？**  
A: 请使用 XMind Zen 或 XMind 2024 版本。旧版可能不兼容，建议导入 Markdown 格式。

**Q: 如何修改指标目标值？**

**Q: 能否导出为图片？**

**Q: 如何添加新指标？**  
A: 在 XMind 中直接添加节点，或在 Markdown 中按层级添加标题。
