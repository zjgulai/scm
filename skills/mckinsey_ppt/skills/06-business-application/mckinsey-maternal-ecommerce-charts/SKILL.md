# 麦肯锡母婴跨境电商图表应用 Skill

## 概述

本Skill将麦肯锡图表方法论应用于母婴跨境电商的具体业务场景，提供可直接复用的图表-观点-建议组合模板。

## 核心理念

```
数据分析 → 图表呈现 → 观点提炼 → 业务建议

数据: 原始数字和指标
图表: 可视化呈现
观点: 核心发现和洞察
建议: 可行动的决策支持
```

## 12大业务场景图表模板

### 场景1: 品类销售分析

**业务问题**: 哪些品类是增长引擎？哪些需要关注？

**推荐图表**: 柱状图+折线图（复合图表）

**数据结构**:
```
品类        销售额(万)  同比增长    贡献度
奶粉        1200       +25%       35%
纸尿裤      800        +15%       23%
辅食        600        +45%       18%
洗护        400        +30%       12%
玩具        300        +20%       9%
服装        150        -5%        3%
```

**幻灯片模板**:
```markdown
# 辅食品类增速领先达45%，建议加大资源投入

## 概述图表
图表展示了各品类销售额与同比增长率的对比分析

## 通过图表可知
• 辅食品类增速最高(+45%)，是增长最快的品类
• 奶粉品类贡献度35%，是销售主力，增速25%稳健
• 服装品类出现负增长(-5%)，需分析原因
• 纸尿裤增速较低(+15%)，市场可能趋于饱和

## 建议行动
• 短期: 增加辅食品类SKU和营销投入
• 中期: 优化奶粉品类利润结构
• 长期: 评估服装品类是否需要调整策略

## 关键词
Category Analysis | Growth Rate | Contribution
```

## 图表-观点-建议组合框架

### 标准框架

```
┌─────────────────────────────────────────────────────────┐
│ 标题 = 核心观点（数据支撑的行动建议）                     │
├─────────────────────────────────────────────────────────┤
│ 概述图表                                                 │
│ 一句话说明图表展示什么数据                                │
├─────────────────────────────────────────────────────────┤
│ 通过图表可知                                             │
│ • 发现1（最重要）- 数据支撑                               │
│ • 发现2 - 数据支撑                                       │
│ • 发现3 - 数据支撑                                       │
├─────────────────────────────────────────────────────────┤
│ 建议行动 / 值得关注                                      │
│ • 行动1（可执行）                                        │
│ • 行动2                                                  │
├─────────────────────────────────────────────────────────┤
│ 关键词                                                   │
│ Keyword1 | Keyword2 | 数据来源                          │
├─────────────────────────────────────────────────────────┤
│ 图表                                                     │
│ [可视化呈现]                                             │
└─────────────────────────────────────────────────────────┘
```

### 观点提炼公式

```
公式1: [主体]在[维度]上[状态/趋势]，建议[行动]
示例: 辅食品类增速领先达45%，建议加大资源投入

公式2: [A]与[B]相比[差异]，这意味着[含义]
示例: TikTok渠道ROI高于亚马逊14%，意味着内容营销效率更高

公式3: 基于[趋势/数据]，预计[结果]，建议[行动]
示例: 基于东南亚55%增速，预计2年内成为第二大市场，建议提前布局

公式4: [发现]表明[问题/机会]，应[行动]
示例: 缺货率5%表明库存管理存在问题，应增加安全库存
```

## 何时不要使用

- 需求仅是简单文案润色或排版修饰，不涉及本 Skill 对应的方法或图表决策。
- 用户问题与该 Skill 的核心场景不匹配，存在更直接的专项 Skill 可用。
- 仅需一次性小修改且不需要此 Skill 的方法论框架。

## 快速使用清单

- 明确页面目标或分析问题（先结论后图表）。
- 确认输入数据字段、时间口径、分组维度。
- 选择图表类型或方法模板并执行。
- 产出标题、关键结论、图表与数据来源。
- 用本 Skill 的质量检查项完成最终自检。

## 常见错误

- 图表类型与业务问题不匹配，导致结论表达失焦。
- 标题不是结论句，只写成章节名。
- 过度堆叠信息，单页承载超过可读上限。
- 缺失数据口径和来源说明，影响复用与可信度。
