# 用户多维度洞察 Skill 参考

## 1. 推荐交付物命名

如无特别要求，建议输出为：

- `*_计算结果.xlsx`
- `*_分析结论.md`
- `*_管理层汇报.pptx`
- `*_数据校验.txt`

## 2. 推荐 Excel Sheet 结构

### 必备 Sheet

1. `指标明细`

## 4. 推荐 PPT 页面模板

### 第 1 页：执行摘要

- 标题：一句完整结论
- 5 个 KPI 卡片
- 3 到 4 条管理层 bullet

### 第 2 页：整体效率

- 图表：整体 KPI 对比
- 结论：短期修复 vs 年度基线差距

### 第 3 页：区域与新老客

- 图表：区域对比 + 新老客对比
- 结论：哪个区域更优、老客价值是否更高

### 第 4 页：资源匹配度

- 图表：四象限 / 气泡图
- 结论：加码对象、优化对象

### 第 5 页：费用结构与行动建议

- 图表：堆叠费用结构
- 结论：哪类费用占比过高、预算动作优先级

## 5. 推荐图表选型

| 分析问题 | 推荐图表 |
|---|---|
| 整体 KPI 对比 | 分组柱状图 |
| 规模与效率同时看 | 双轴柱线图 |
| 区域 / 新老客对比 | 分组柱状图 |
| 投入占比 vs 产出占比 | 四象限散点 / 气泡图 |
| 三费结构 | 横向堆叠条形图 |

## 6. 校验规则

### 汇总校验

- `整体-新客 + 整体-老客 = 整体-整体`
- 区域整体汇总应等于整体整体

### 公式校验

- `人均毛利率 = 人均毛利额 / 人均销售额`
- `人均订单价 = 人均销售额 / 人均订单数`
- `人均客品数 = 销量 / 订单数`

### 容差建议

对 Excel 浮点与四舍五入误差，建议使用 `0.05` 以内容差，不要把 0.004 级别误差判成失败。
