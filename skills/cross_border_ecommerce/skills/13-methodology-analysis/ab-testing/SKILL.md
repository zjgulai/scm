# 跨境电商A/B测试分析

## 一、Skill概述

A/B测试是通过随机对照实验，验证某个改动是否有效的科学方法。通过对比实验组和对照组的表现，可以建立因果关系，判断改动是否带来了真实的提升。

**核心价值**：
- 科学验证改动效果
- 建立因果关系
- 降低决策风险
- 持续优化产品

### 2.2 核心概念

| 概念 | 说明 |
|------|------|
| **实验组** | 应用新方案的群体 |
| **对照组** | 保持原方案的群体 |

| **统计显著性** | 结果差异不太可能是偶然的 |

## 三、三类典型实验设计

### 3.1 页面类实验

**场景**：优化页面布局、文案、UI元素

```
示例：商品详情页优化

问题：排行榜功能点击率高，但转化率低
假设：调整商品排序可以提升转化率

实验设计：
├── 对照组（A版）：现有排序
├── 实验组（B版）：热销商品前置
│
├── 北极星指标：排行榜路径成交率
├── 过程指标：详情页→购物车转化率
└── 护栏指标：页面加载时间、跳出率
```

**页面类实验要点**：
- 单变量原则：每次只改一个元素
- 视觉一致性：确保不同版本体验一致
- 兼容性测试：检查不同设备/浏览器

### 3.2 优惠类实验

**场景**：测试不同优惠策略的效果

```
示例：优惠券面额测试

问题：优惠券使用率低
假设：调整优惠力度可以提升使用率

实验设计：
├── A组：满100减10元
├── B组：满100减15元
├── C组：满100减20元
│
├── 北极星指标：GMV增量
├── 过程指标：优惠券使用率、客单价
└── 护栏指标：毛利率、ROI
```

**优惠类实验要点**：
- 控制用户可见性：避免用户对比产生不满
- 考虑成本：计算增量收益是否覆盖成本
- 防止薅羊毛：设置合理的使用门槛

### 3.3 策略类实验

**场景**：测试算法、推荐策略、运营策略

```
示例：推荐算法测试

问题：推荐点击率下降
假设：新算法可以提升推荐效果

实验设计：
├── 对照组：现有推荐算法
├── 实验组：新推荐算法
│
├── 北极星指标：推荐带来的GMV
├── 过程指标：推荐点击率、加购率
└── 护栏指标：页面加载速度、推荐多样性
```

**策略类实验要点**：
- 样本量计算：策略效果通常较小，需要更大样本
- 长期效应：考虑学习效应和新鲜感效应
- 分层分析：不同用户群体可能效果不同

## 四、统计显著性判断

### 4.1 样本量计算

```
样本量取决于：
├── 基准转化率（当前水平）
├── 最小检测效应（希望检测的最小提升）
├── 显著性水平（α，通常0.05）
└── 统计功效（1-β，通常0.8）

公式简化版：
n = 16 × σ² / Δ²

其中：
σ = 标准差
Δ = 期望检测的最小差异
```

### 4.2 显著性检验

```
假设检验框架：
├── 原假设（H0）：两组无差异
├── 备择假设（H1）：两组有差异
│
├── p值 < 0.05：拒绝原假设，结果显著
└── p值 ≥ 0.05：无法拒绝原假设，结果不显著
```

### 4.3 置信区间

```
转化率差异的95%置信区间：
CI = 差异 ± 1.96 × SE

如果置信区间不包含0，则差异显著
```

## 五、常见问题与解决方案

### 5.1 为什么实验有效，上线后没效果？

**可能原因**：
1. **新奇效应**：用户因为新鲜而点击，不代表长期效果
2. **样本偏差**：实验期间的用户与全量用户不同
3. **季节性因素**：实验期间的季节性影响了结果
4. **交互效应**：实验组可能与其他功能产生交互

**解决方案**：
- 延长实验周期，观察长期效果
- 分层分析，检查不同群体效果
- 考虑季节性因素，选择合适时机

### 5.2 样本量不够怎么办？

**替代方案**：
1. **事后分析**：使用历史数据对比
2. **分层匹配**：找到相似的对照组
3. **时间序列分析**：对比改动前后趋势
4. **小流量长周期**：用时间换空间

### 5.3 多指标如何决策？

**决策原则**：
1. 北极星指标优先
2. 考虑护栏指标的红线
3. 综合评估过程指标
4. 权衡短期vs长期效果

## 九、注意事项

1. **单一变量原则**：每次只测试一个改动
2. **样本量充足**：确保有足够的统计功效
3. **随机分流**：确保组间可比性
4. **避免窥视**：不要在实验中途做决策
5. **护栏指标**：防止负面影响
6. **长期视角**：考虑长期效应而非短期
