## 本地重建说明

本节为基于当前项目本地资料重建的仓网规划执行方案，不等同于钉钉原文复制。重建依据来自 `ref/books/供应链36%方案_Page6节点/09-重点节点02-仓网规划执行方案.md` 以及本专题 Data / Plan / Report 层。

## 1. 节点定位

仓网规划负责让“位置正确”。它不是仓租优化，而是库存位置、成本、时效和风险之间的结构性决策。

## 2. 执行目标

| 目标 | 管理含义 | 验收指标 |
|---|---|---|
| 明确仓型角色 | 避免所有仓承担同一职能 | 本地履约仓、跨境补货仓、逆向处理仓清单 |
| 修正区域库存结构 | 降低北美、拉美、中东等区域风险 | 区域库存周转天数、超龄库存占比 |
| 建立调拨候选池 | 让调拨从救火变成机制 | 调拨成功率、调拨 ROI |
| 联动尾程成本 | 仓网调整必须改善服务半径 | 尾程成本率、配送时效 |

## 3. 输入字段

| 数据域 | 字段 | 来源 | 用途 |
|---|---|---|---|
| 区域库存 | 区域、国家、仓、SKU、库存金额、库龄 | `dwt_inventory_health` | 区域风险图 |
| 销售需求 | 国家/渠道/GTM 销售额、销量、预测需求 | `dwt_supply_chain_cost` + 预测版本 | 仓网配置 |
| 履约表现 | 尾程成本、时效、妥投率、异常件率 | `dwt_fulfillment_stability` | 服务半径评估 |
| 仓储成本 | 仓租、操作费、长期仓储费 | WMS/财务 | 仓型成本评估 |
| 调拨 | 源仓、目标仓、调拨成本、到货时效 | WMS/TMS | 调拨收益测算 |

## 4. 指标与阈值

| 指标 | 公式/口径 | 阈值 | Owner |
|---|---|---|---|
| 仓网覆盖率 | `有库存国家仓组合/计划组合` | <85% 提示 | 仓网规划负责人 |
| 区域库存周转天数 | 区域库存 / 区域日均 COGS | >100天预警 | 仓网规划负责人 |
| 超龄库存占比 | `90天+库存金额/总库存金额` | >10% 预警 | 库存计划 |
| 单仓周转天数 | 单仓库存 / 单仓日均 COGS | >120天预警 | 仓储经理 |
| 调拨成功率 | `调拨完成/调拨建议` | <80% 提示 | 调拨经理 |
| 调拨后缺货下降率 | 调拨后缺货下降幅度 | <10% 复盘 | 调拨经理 |

## 5. 执行 SOP

| 节奏 | 动作 | 输出 |
|---|---|---|
| 周度 | 输出区域库存热力图、超龄仓清单、低覆盖国家清单 | 仓网风险周报 |
| 周度 | 生成冗余仓到缺货仓候选调拨池 | 调拨候选单 |
| 月度 | 复盘区域周转、尾程成本、时效变化 | 区域仓网复盘 |
| 季度 | 重评估仓型、仓租、服务半径、销量迁移 | 仓网策略版本 |

## 6. 红线规则

| 条件 | 动作 |
|---|---|
| 单仓周转 >120天 | 进入清仓/暂停补货专题 |
| 单国铺货率 <60% | 禁止以压仓租为由关仓 |
| 拉美继续恶化 | 暂停新增补货并启动清库机制 |
| 北美周转连续两周拉升 | 启动北美-欧洲分流复盘 |

## 7. 区域策略

| 区域 | 策略 | 依据 |
|---|---|---|
| 北美 | 主战区复合治理，联动采购、尾程、仓储 | 成本率高、库存压力高 |
| 欧洲 | 第二主仓与弹性调拨承接 | 成本率相对健康，可承接调拨 |
| 拉美 | 暂停补货 + 清库优先 | 周转恶化 |
| 中东 | 风险仓策略，控制超龄 | 超龄压力高 |
| 亚太 | 轻仓试点，小步扩仓 | 周转相对健康 |

## 8. 阶段目标

| 时间 | 目标 |
|---|---|
| 60天 | 完成区域风险地图和仓型角色定义 |
| 90天 | 北美/欧洲调拨试点落地，拉美清库机制运行 |
| 180天 | 形成季度仓网重评估机制，区域超龄和周转风险收敛 |

## 9. 节点接口

| 接口 | 内容 |
|---|---|
| 接收 `kp01-需求预测` | 区域需求变化趋势 |
| 接收 `kp03-计划排产` | 补货节奏约束 |
| 输出给 `kp04-仓储与调拨` | 区域候选调拨池 |
| 输出给履约尾程 | 服务半径与履约档位策略 |
