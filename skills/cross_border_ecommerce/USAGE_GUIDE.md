# 母婴跨境电商Claude Skills使用指南

> **版本**: 3.0 (母婴增强版) | **更新日期**: 2024-01
> **适用对象**: 母婴跨境电商运营、数据分析、市场投放、供应链管理人员
> **核心特性**: 母婴行业专业化、宝宝生命周期分析、品类迁移追踪、订阅制优化

## 🎯 v3.0 母婴增强版新特性

1. **母婴行业知识注入**: 所有Skills深度融入母婴行业特性

6. **节日日历**: 母婴专属节日需求预测(圣诞节、儿童节等)

## 快速开始

### 如何触发Skills

在Claude对话中使用以下方式触发Skills：

**方式1: 关键词触发**
```
用户: "帮我分析客户RFM分群"
Claude: [自动加载cbec-customer-segmentation Skill]
```

**方式2: 场景描述触发**
```
用户: "最近销售额下降了，想知道是什么原因"
Claude: [自动加载cbec-contribution-calculation Skill]
```

**方式3: 上传数据后触发**
```
用户: [上传销售数据] "请分析这个月的销售趋势"
Claude: [自动加载cbec-yoy-mom-analysis Skill]
```

### 触发词优先级

当多个触发词同时出现时，按以下优先级匹配：
1. 精确匹配（如"RFM分群"→customer-segmentation）
2. 场景匹配（如"为什么下降"→归因分析）
3. 通用匹配（如"分析"→data-analyst）

#### 7.5 cbec-complex-contribution (复杂贡献度分析) ✨NEW

| 触发词 | 示例 |
|--------|------|
| 复杂贡献度 | "分析复杂指标的贡献度" |
| 偏微分贡献 | "用偏微分计算贡献" |
| 全微分归因 | "全微分归因分析" |
| 多因素贡献 | "多因素变动贡献" |
| 四则运算贡献 | "加减乘除混合的贡献分析" |
| 弹性系数 | "计算各因素弹性系数" |
| 敏感度分析 | "分析因素敏感度" |

**能解决的决策问题**:
- ✅ 指标关系包含加减乘除混合运算时如何归因？
- ✅ 各因素对结果的敏感度（弹性系数）是多少？
- ✅ 各因素变动对总变动的精确贡献占比？
- ✅ 因素间的交互效应有多大？

**支持的指标关系类型**:
| 类型 | 公式示例 | 偏导数特点 |
|------|---------|-----------|
| 乘法链式 | Y = A × B × C | ∂Y/∂A = B × C |
| 减法链式 | Y = 1 - A - B - C | ∂Y/∂A = -1 |
| ROI型 | Y = (A-B)/B | ∂Y/∂B = -A/B² |
| 混合运算 | Y = (A×B+C)/D | 多元非线性 |
| 自定义 | 任意四则运算 | 自动计算 |

**核心算法**:
```
静态贡献度（敏感度）= 偏导数值 ∂Y/∂Xᵢ
弹性系数 = (∂Y/∂Xᵢ) × (Xᵢ/Y)
波动贡献度 = (∂Y/∂Xᵢ) × ΔXᵢ
贡献占比 = 波动贡献度 / 总变动 × 100%
交互效应 = 实际变动 - 一阶全微分合计
```

### 📈 趋势分析层 (1个Skill)

#### 10. cbec-yoy-mom-analysis (同比环比分析)

| 触发词 | 示例 |
|--------|------|
| 同比分析、YoY | "同比增长率分析" |
| 环比分析、MoM | "环比分析" |
| 增长率 | "计算增长率" |
| 趋势分析 | "分析趋势变化" |
| CAGR | "计算复合增长率" |
| MAT | "滚动年度分析" |
| YTD | "年初至今分析" |

**能解决的决策问题**:
- ✅ 增长趋势是否健康？
- ✅ 与去年同期相比表现如何？
- ✅ 短期波动还是长期趋势？
- ✅ 季节性规律是什么？

**核心公式**:
```
同比增长率(YoY) = (本期值 - 去年同期值) / 去年同期值 × 100%
环比增长率(MoM) = (本期值 - 上期值) / 上期值 × 100%
复合年增长率(CAGR) = (期末值/期初值)^(1/年数) - 1
```

## 常见问题解答

### Q1: 如何知道该用哪个Skill？

**回答**: 根据你的问题类型选择：
- 问"为什么"→ 归因分析类
- 问"多少"→ 预测分析类
- 问"怎么样"→ 趋势分析类
- 问"分几类"→ 分群分析类
- 问"好不好"→ 评估分析类

### Q2: 多个Skills如何配合使用？

**回答**: 按工作流顺序调用：
1. 先用`cbec-data-cleaning`确保数据质量
2. 再用分析类Skill得出结论
3. 最后用`cbec-knowledge-query`获取最佳实践

### Q3: 数据格式有什么要求？

**回答**:
- 支持CSV、Excel、JSON格式
- 必须包含必要字段（如日期、金额、ID等）
- 建议先用`cbec-data-cleaning`清洗数据

### Q4: 母婴行业有什么特殊考虑？

**回答**:
- 客户生命周期与宝宝成长相关
- 需要关注产品安全性和合规
- 妈妈群体重视口碑和社区
- 价格敏感但愿意为品质付费
- 决策周期长，需要多次触达

### Q5: 如何获取行业最佳实践？

**回答**: 使用`cbec-knowledge-query` Skill：
```
"查询母婴行业客户运营最佳实践"
"查询跨境电商定价策略"
"查询库存管理知识"
```
