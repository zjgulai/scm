# 跨境电商杜邦财务分析 Skill

## 概述

杜邦分析法(DuPont Analysis)是财务分析的经典框架，通过分解净资产收益率(ROE)来系统评估企业的盈利能力、资产运营效率和财务杠杆。本Skill专为跨境电商企业设计，考虑行业特性（轻资产、高周转、多币种），提供标准化的财务诊断能力。

### 为什么跨境电商需要杜邦分析？

1. **多维度评估**: 不仅看利润，还要看资产效率和资本结构
2. **问题定位**: 快速识别ROE变动的驱动因素
3. **对标分析**: 与行业标杆对比，找到差距
4. **战略指导**: 为提升ROE提供明确方向

## 核心功能

### 1. 三因素杜邦分析

**经典公式**:
```
ROE = 净利润率 × 总资产周转率 × 权益乘数

其中:
- 净利润率 = 净利润 / 营业收入
- 总资产周转率 = 营业收入 / 总资产
- 权益乘数 = 总资产 / 股东权益
```

**业务含义**:
| 因素 | 指标 | 跨境电商典型值 | 优化方向 |
|------|------|---------------|---------|
| 盈利能力 | 净利润率 | 5%-15% | 提升客单价、降低成本 |
| 运营效率 | 资产周转率 | 2-5次/年 | 加速库存周转 |
| 财务杠杆 | 权益乘数 | 1.5-3.0 | 适度负债，控制风险 |

### 2. 五因素扩展分析

**扩展公式**:
```
ROE = 税收负担 × 利息负担 × 营业利润率 × 资产周转率 × 权益乘数

其中:
- 税收负担 = 净利润 / 税前利润
- 利息负担 = 税前利润 / EBIT
- 营业利润率 = EBIT / 营业收入
- 资产周转率 = 营业收入 / 总资产
- 权益乘数 = 总资产 / 股东权益
```

**应用场景**: 当净利润率异常时，进一步拆解是税负、利息还是经营问题。

### 3. 多期趋势分析

- 追踪ROE及各因素的历史变化
- 识别趋势和异常点
- 预测未来走向

### 4. 同行业对比

- 与行业标杆对比
- 识别竞争优势和劣势
- 设定改进目标

## 使用方法

### 输入数据要求

**必需财务报表**:

1. **利润表** (Income Statement)
   - 营业收入 (Revenue)
   - 营业成本 (COGS)
   - 营业费用 (Operating Expenses)
   - 营业利润 (Operating Income/EBIT)
   - 利息费用 (Interest Expense)
   - 税前利润 (EBT)
   - 所得税 (Tax)
   - 净利润 (Net Income)

2. **资产负债表** (Balance Sheet)
   - 流动资产 (Current Assets)
     - 现金及等价物
     - 应收账款
     - 存货
   - 非流动资产 (Non-current Assets)
   - 总资产 (Total Assets)
   - 流动负债 (Current Liabilities)
   - 非流动负债 (Non-current Liabilities)
   - 总负债 (Total Liabilities)
   - 股东权益 (Shareholders' Equity)

**数据格式**:
- 支持CSV、Excel格式
- 支持多期数据（建议至少4个季度或3年）
- 支持多币种（自动转换）

### 分析流程

```
步骤1: 数据加载与验证
- 读取财务报表数据
- 验证数据完整性
- 检查会计恒等式（资产=负债+权益）

步骤2: 计算核心指标
- 计算净利润率、资产周转率、权益乘数
- 计算五因素分解指标
- 计算辅助指标（毛利率、费用率等）

步骤3: 杜邦分解
- 构建杜邦分析金字塔
- 计算各因素贡献度
- 识别主要驱动因素

步骤4: 趋势分析
- 多期指标对比
- 变化趋势可视化
- 异常点识别

步骤5: 对标分析
- 与行业均值对比
- 与标杆企业对比
- 差距分析

步骤6: 生成报告
- 杜邦分析金字塔图
- 详细分析报告
- 改进建议
```

## 算法说明

### ROE分解计算

**基础公式**:
```python
roe = net_income / shareholders_equity

net_profit_margin = net_income / revenue
asset_turnover = revenue / total_assets
equity_multiplier = total_assets / shareholders_equity

# 验证
roe == net_profit_margin * asset_turnover * equity_multiplier
```

**因素变动贡献**:
```
ROE变动 = Δ净利润率贡献 + Δ周转率贡献 + Δ杠杆贡献

Δ净利润率贡献 = (本期净利率 - 上期净利率) × 上期周转率 × 上期杠杆
Δ周转率贡献 = 本期净利率 × (本期周转率 - 上期周转率) × 上期杠杆
Δ杠杆贡献 = 本期净利率 × 本期周转率 × (本期杠杆 - 上期杠杆)
```

### 财务健康评分

基于杜邦指标计算综合评分:
```
评分 = w1×净利率得分 + w2×周转率得分 + w3×杠杆得分

其中权重: w1=0.4, w2=0.35, w3=0.25

得分计算（以净利率为例）:
- 净利率 ≥ 15%: 100分
- 净利率 10%-15%: 80分
- 净利率 5%-10%: 60分
- 净利率 < 5%: 40分
```

## 示例

### 示例1: 单期杜邦分析

**输入数据** (年度财务报表):
```
利润表关键数据:
- 营业收入: $10,000,000
- 净利润: $800,000

资产负债表关键数据:
- 总资产: $5,000,000
- 股东权益: $2,500,000
```

**计算过程**:
```
净利润率 = 800,000 / 10,000,000 = 8%
资产周转率 = 10,000,000 / 5,000,000 = 2次
权益乘数 = 5,000,000 / 2,500,000 = 2

ROE = 8% × 2 × 2 = 32%
```

**分析解读**:
- ROE为32%，高于行业平均(25%)
- 主要驱动因素是资产周转率(2次)，说明资产利用效率高
- 净利润率(8%)有提升空间，建议优化成本结构

### 示例2: 多期趋势分析

**数据** (3年财务数据):
| 年度 | ROE | 净利率 | 周转率 | 杠杆 |
|------|-----|--------|--------|------|
| 2022 | 28% | 10% | 1.8 | 1.56 |
| 2023 | 30% | 9% | 2.0 | 1.67 |
| 2024 | 25% | 8% | 1.9 | 1.65 |

**趋势分析**:
- ROE从30%下降到25%，需关注
- 净利率持续下降（10%→8%），是主要问题
- 周转率提升后回落，需稳定运营效率

**因素贡献分析**:
```
2023→2024 ROE变动: 25% - 30% = -5%

净利率贡献: (8%-9%) × 2.0 × 1.67 = -3.34%
周转率贡献: 8% × (1.9-2.0) × 1.67 = -1.34%
杠杆贡献: 8% × 1.9 × (1.65-1.67) = -0.30%

验证: -3.34% + (-1.34%) + (-0.30%) ≈ -5%
```

**结论**: ROE下降主要由净利率下降导致，需重点优化盈利能力。

### 示例3: 同行业对比

**本公司 vs 行业标杆**:
| 指标 | 本公司 | 行业标杆 | 差距 |
|------|--------|----------|------|
| ROE | 25% | 30% | -5% |
| 净利率 | 8% | 12% | -4% |
| 周转率 | 1.9 | 1.8 | +0.1 |
| 杠杆 | 1.65 | 1.39 | +0.26 |

**分析**:
- 优势: 资产周转效率略高于标杆
- 劣势: 净利率明显低于标杆，是主要差距来源
- 风险: 杠杆率高于标杆，财务风险较大

**建议**:
1. 提升净利率: 优化产品结构、控制成本
2. 降低杠杆: 适度降低负债，优化资本结构

## 注意事项

### 数据质量
- 确保财务报表数据准确完整
- 注意会计政策一致性
- 多币种需统一汇率

### 行业差异
- 不同跨境电商模式（平台卖家、独立站）财务特征不同
- 对比时应选择同类型企业

### 局限性
- 杜邦分析基于历史财务数据，不反映未来
- 不考虑非财务因素（品牌价值、市场份额等）
- 对轻资产互联网企业适用性有限

### 最佳实践
1. **定期分析**: 每季度进行杜邦分析，追踪变化
2. **多维对比**: 与历史、行业、标杆多维度对比
3. **结合业务**: 财务数据与业务数据结合分析
4. **行动导向**: 分析结果转化为具体改进行动

## 高级功能

### 情景模拟

模拟不同情景下的ROE变化:
```
情景1: 净利率提升2%
情景2: 周转率提升0.5次
情景3: 同时优化净利率和周转率
```

### 敏感性分析

分析各因素对ROE的敏感程度:
```
敏感系数 = ROE变化% / 因素变化%

例: 净利率敏感系数 = 2.5
意味着净利率每提升1%，ROE提升2.5%
```

### 多实体对比

支持同时分析多个子公司/品牌:
- 集团层面汇总分析
- 各子公司对比分析
- 识别最佳实践
