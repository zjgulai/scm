# 供应商绩效分析 (Supplier Performance Analysis)

## 概述

供应商绩效分析是供应链管理中至关重要的一环，它不仅影响企业的采购决策，还直接关系到产品质量和成本控制。有效的供应商评估可以帮助企业识别合适的合作伙伴，降低风险，提高供应链的整体效率。

## 核心功能

### 1. 供应商评估

多维度评估供应商：
- **质量维度**: 产品合格率、质量投诉率
- **交付维度**: 准时交付率、交货准确性
- **成本维度**: 采购成本、成本变动率
- **服务维度**: 响应时间、订单处理效率
- **创新维度**: 新品开发能力、改进建议

### 2. 供应商分级管理

对供应商进行分类分级：
- 战略供应商
- 优选供应商
- 资格未定供应商
- 被动淘汰供应商
- 主动淘汰供应商

### 3. 供应商优化

制定优化策略：
- 绩效改进计划
- 战略合作深化
- 供应商淘汰机制

## 使用方法

### 输入要求

| 数据类型 | 必需字段 | 说明 |
|---------|---------|------|
| 基础信息 | supplier_id, name, category | 供应商标识 |
| 质量数据 | defect_rate, complaint_count | 质量指标 |
| 交付数据 | on_time_rate, lead_time | 交付指标 |
| 成本数据 | unit_cost, cost_trend | 成本指标 |
| 服务数据 | response_time, issue_resolve_rate | 服务指标 |

### 分析流程

```
1. 数据收集
   ├── 采购系统数据
   ├── 质检数据
   ├── 财务数据
   └── 业务反馈

2. 指标计算
   ├── 质量指标 (合格率、投诉率)
   ├── 交付指标 (准时率、提前期)
   ├── 成本指标 (价格、变动)
   └── 服务指标 (响应、解决率)

3. 综合评分
   ├── 设定权重
   ├── 计算得分
   └── 排名分析

4. 分级管理
   ├── 确定分级标准
   ├── 供应商分类
   └── 制定管理策略

5. 输出报告
   ├── 绩效评估表
   ├── 分级建议
   └── 优化方案
```

## 核心公式

### 综合绩效评分

```
综合得分 = w1×质量分 + w2×交付分 + w3×成本分 + w4×服务分

推荐权重:
- 质量: 35%
- 交付: 30%
- 成本: 20%
- 服务: 15%
```

### 质量得分

```
质量得分 = 100 - (不合格率 × 100) - (投诉率 × 50)
```

### 交付得分

```
交付得分 = 准时交付率 × 100

准时交付率 = 准时订单数 / 总订单数 × 100%
```

### 成本竞争力

```
成本竞争力 = (市场均价 - 供应商价格) / 市场均价 × 100
```

## 示例

### 示例1: 供应商年度评估

**场景**: 某母婴跨境电商年度供应商评估

**评估数据**:
| 供应商 | 类别 | 质量分 | 交付分 | 成本分 | 服务分 | 综合分 |
|-------|------|--------|--------|--------|--------|--------|
| A工厂 | 奶瓶 | 98 | 95 | 85 | 92 | 94 |
| B工厂 | 奶瓶 | 92 | 88 | 90 | 85 | 89 |
| C工厂 | 纸尿裤 | 95 | 92 | 88 | 88 | 91 |
| D工厂 | 玩具 | 85 | 78 | 92 | 80 | 83 |

**分级结果**:
- A工厂 → 战略供应商 (核心品类，深度合作)
- B工厂 → 优选供应商 (备选供应商)
- C工厂 → 战略供应商 (主力供应商)
- D工厂 → 资格未定 (需改进交付)

**优化建议**:
1. 与A工厂签订长期战略合作协议
2. 帮助D工厂改进交付流程
3. 增加B工厂订单占比，分散风险

### 示例2: 新供应商准入评估

**场景**: 评估是否引入新供应商

**评估标准**:
| 维度 | 指标 | 权重 | 及格线 |
|-----|------|------|--------|
| 质量 | 样品合格率 | 35% | ≥95% |
| 交付 | 样品交付准时率 | 25% | ≥90% |
| 成本 | 价格竞争力 | 25% | 低于现供应商10% |
| 资质 | 认证证书 | 15% | 必须有ISO认证 |

**评估结果**:
```
新供应商E评估:
├── 质量得分: 96分 (样品合格率98%)
├── 交付得分: 90分 (准时率92%)
├── 成本得分: 88分 (价格低12%)
├── 资质得分: 95分 (ISO9001认证)
└── 综合得分: 93分 → 通过准入
```

## 注意事项

### 数据层面
1. **数据准确性**: 确保采购、质检数据准确
2. **统计周期**: 使用合理的统计周期(季度/年度)
3. **异常处理**: 排除不可抗力导致的异常

### 分析层面
1. **权重设定**: 根据业务特点调整权重
2. **动态评估**: 定期更新评估结果
3. **横向对比**: 与行业标杆对比

### 业务层面
1. **战略合作**: 对战略供应商投入更多资源
2. **风险分散**: 避免过度依赖单一供应商
3. **双向沟通**: 建立供应商沟通机制
