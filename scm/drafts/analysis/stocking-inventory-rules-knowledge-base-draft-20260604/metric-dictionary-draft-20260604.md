# 备货库存规则指标字典

## 编码规则

备货库存指标使用独立编码，不占用原积加 SCM 指标编码。

```text
STOCK-[DOMAIN]-[NNN]
```

| 域 | 含义 |
|---|---|
| `BIZ` | 业务库存通用指标 |
| `PLAN` | 计划库存和采购未交 |
| `INTRANSIT` | 在途库存 |
| `PLATFORM` | 平台库存 |
| `MAP` | 映射质量 |
| `SYNC` | 同步与刷新 |

## 通用业务库存指标

| 指标编码 | 指标名称 | 口径 | 粒度 | 一致性约束 |
|---|---|---|---|---|
| STOCK-BIZ-001 | 可用库存 | 可用于备货判断的可售或可用数量，按平台规则取源字段或派生 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 对齐积加 `wh_available_qty`，但保留平台公式 |
| STOCK-BIZ-002 | 预占库存 | 被订单、买家订单或业务处理占用的数量 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 对齐积加 `wh_reserved_qty` |
| STOCK-BIZ-003 | 冻结库存 | 规则定义为冻结或运营中心处理中但仍属良品范围的数量 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 新增独立指标，不覆盖积加字段 |
| STOCK-BIZ-004 | 不良品库存 | 次品、不可售、拒收损坏或平台不良品数量 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 对齐积加 `wh_defective_qty` |
| STOCK-BIZ-005 | 在库库存 | 可用库存 + 预占库存 + 冻结库存 + 不良品库存 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 业务派生，不覆盖积加 `wh_onhand_qty` |
| STOCK-BIZ-006 | 在库良品库存 | 可用库存 + 预占库存 + 冻结库存 | 渠道 + 业务主键 + 仓库/国家 + 快照时间 | 业务派生，不覆盖积加良品量 |

## 计划与采购指标

| 指标编码 | 指标名称 | 口径 | 粒度 | 来源 |
|---|---|---|---|---|
| STOCK-PLAN-001 | 计划库存 | 按平台规则统计未来将进入目标渠道或仓库的计划性库存 | 平台规则指定粒度 | SRM 需求池、SRM 采购订单、调拨单 |
| STOCK-PLAN-002 | 采购未交库存 | 采购订单明细和送货单明细组合，等于在途数量、未交货数量、送货单明细发货数量合并 | 仓库 + 供应链 SKU | SRM 采购订单、送货单 |
| STOCK-PLAN-003 | 待订单需求库存 | SRM 需求池待订单状态数量 | 仓库 + 供应链 SKU | SRM 需求池 |
| STOCK-PLAN-004 | 采购订单计划库存 | SRM 采购订单新建、变更中、待供应商确认、供应商拒绝、易仓处理中状态数量 | 仓库 + 供应链 SKU | SRM 采购订单 |

## 在途指标

| 指标编码 | 指标名称 | 口径 | 粒度 | 来源 |
|---|---|---|---|---|
| STOCK-INTRANSIT-001 | FBA 在途库存 | FBA 货件指定状态数量，且关联调拨单为调拨在途或已完成 | 仓库 + MSKU | FBA 货件、调拨单 |
| STOCK-INTRANSIT-002 | Walmart 在途库存 | 物流跟踪表状态为正常在途、出口查验未放行、进口查验未放行、延误风险的 SKU 明细数量 | ItemID + 国家 | 自研 ERP 物流跟踪表 |
| STOCK-INTRANSIT-003 | TikTok 在途库存 | 物流跟踪表按 TikTok 渠道和目的国家汇总的 SKU 明细数量 | MSKU + 国家 | 自研 ERP 物流跟踪表 |
| STOCK-INTRANSIT-004 | Shopify 在途库存 | 调拨单关联发货单，发货单状态为提货中、已出运、入库中时的发货量 | 供应链 SKU + 仓库 | 调拨单、发货单 |

## 映射与同步指标

| 指标编码 | 指标名称 | 口径 | 用途 |
|---|---|---|---|
| STOCK-MAP-001 | 平台主键映射扩展倍数 | 一个源 SKU/GTIN 映射到的平台主键数量 | 识别库存放大风险 |
| STOCK-MAP-002 | 未匹配计划库存数量 | 因缺少 GTIN、MSKU 或销售映射无法匹配的计划库存 | 识别丢失风险 |
| STOCK-SYNC-001 | 库存同步时间 | 页面显示的上次更新完成时间 | 判断库存新鲜度 |
| STOCK-SYNC-002 | 同步延迟分钟数 | 当前时间与来源同步时间差 | 数据质量监控 |

## 统一派生公式

```text
在库库存 = 可用库存 + 预占库存 + 冻结库存 + 不良品库存
在库良品库存 = 可用库存 + 预占库存 + 冻结库存
```

公式适用范围：

- 只适用于备货业务库存。
- 不替代积加 ERP 原生在库量。
- 所有派生结果必须保留来源平台、业务主键、仓库或国家。
