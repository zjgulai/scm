# 全局数据资源整合 · 产出说明

本目录为《全局数据资源整合》子计划的完整产出，供**数仓开发**与**业务/分析**对齐使用。

## Excel 汇总（数仓对接用）

为便于与数仓对接，已将上述文档中**涉及字段与表格的内容**导出为同一工作簿的多 Sheet Excel 文件，便于查阅与批注：

- **文件名**：`全局数据资源整合_数仓对接.xlsx`
- **Sheet 构成**：01_专题课题_数据需求矩阵、02_存量BI指标映射、02_需补充明细字段、03_外部数据需求清单、04_公共维度/指标/专题间复用场景、05_审查结论/事实表/维度表/贴源层。
- **重新生成**：若 Markdown 文档有更新，可在本目录下执行 `python3 export_to_excel.py`（依赖 `pandas`、`openpyxl`）重新生成该 Excel。

## 文档清单与用途

| 文档 | 用途 | 主要读者 |
|------|------|----------|
| **01_专题课题_数据需求矩阵.md** | 4 大专题 × 14 子课题 × 所需字段/英文名/库表建议；与 05 配套为**数仓需求基线** | 数仓、产品、分析 |
| **02_字段对照_存量BI映射表.md** | 通用指标池与存量 BI 指标字典的映射；待补齐字段列表 | 数仓、BI、分析 |
| **03_外部数据需求清单.md** | 专题①③④ 所需外部数据源、最小字段集、采集方式与频率建议 | 数据采集、产品、分析 |
| **04_专题间字段复用说明.md** | 公共维度/指标、跨专题复用场景、口径统一注意事项 | 数仓、BI、分析 |
| **05_数仓表结构与主键设计.md** | 每张 fact/dim/ods 的**粒度、主键（含联合主键）、分区键、逻辑外键**；数仓审查后的表结构规范 | 数仓（必读） |

## 审查与迭代状态

- **最后审查**：从专业数仓视角对主键、联合主键、表粒度、分区、跨表关联做了完整性审查，并完成一轮迭代。
- **迭代内容**：新增 **05_数仓表结构与主键设计.md**，对 01 中建议的每张表补充粒度、主键、分区及关联关系；01 的「数仓开发说明」已指向 05，避免建表时主键/粒度歧义。
- **使用顺序建议**：数仓排期时以 **01 + 05** 为需求与表结构基线，02 用于与现有 BI 映射，03 用于外部数据排期，04 用于公共层与口径统一。

## 与主项目关系

- 本子计划隶属主项目《数智决策-AI效能作战室-产品与项目规划》的 **Phase 0 / 数据输入层**。
- 完成本目录下的数据准备与表结构落地后，分析侧与 5+1 智能体即可按 01 中的「所需核心指标/字段」无阻碍消费数仓产出。
