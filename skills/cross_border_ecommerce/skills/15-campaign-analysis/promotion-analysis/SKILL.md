# 跨境电商促销活动分析

## 一、Skill概述

促销活动分析是评估活动效果、优化活动策略的核心技能。本Skill提供活动全流程分析方法，帮助分析师准确评估活动增量效果，提炼活动经验。

**核心价值**：
- 科学评估活动真实效果
- 区分增量与存量
- 提炼可复用的活动经验
- 优化活动策略

---

## 二、活动本质与分类

### 2.1 什么是活动

```
定义：在常规销售、运营流程以外，额外增加投入，换取额外效果

常规流程：用户进门 → 选商品 → 交钱 → 走人
活动：今天到店消费的用户，可以免费领一个冰淇淋
```

### 2.2 活动分类矩阵

```
活动分类
├── 按对象分
│   ├── 品牌类（客户不付钱）
│   │   └── 纯宣传，不考核业务效果
│   └── 效果类（客户付钱）
│       └── 需要考核业务转化
│
├── 按目标分
│   ├── 人（用户/员工/经销商）
│   ├── 货（商品）
│   └── 场（渠道/区域）
│
└── 按投放分
    ├── 公开（所有用户可见）
    └── 精准（定向投放）
```

### 2.3 活动类型与考核要点

| 活动类型 | 特点 | 考核要点 |
|---------|------|---------|
| **品牌类** | 纯宣传 | 认链接不认说法 |
| **用户类** | 针对消费者 | 增量GMV、复购率 |
| **商品类** | 针对商品 | 商品销量、毛利率 |
| **渠道类** | 针对渠道 | 渠道增量、ROI |
| **经销商类** | 针对渠道商 | 提货量、窜货风险 |

## 四、增量效果评估

### 4.1 增量定义的关键

```
活动需要的是增量，增量定义是关键

对比维度：
├── 有活动的时间 vs 没活动的时间
├── 有活动的商品 vs 没活动的商品
├── 有活动的人 vs 没活动的人
└── 有活动的店 vs 没活动的店
```

### 4.2 增量计算方法

#### 方法一：时间对比法
```
增量 = 活动期销量 - 基准期销量

基准期选择：
- 活动前同期（剔除季节性）
- 去年同期（控制周期性）
- 历史平均值（平滑波动）
```

#### 方法二：对照组法
```
增量 = 活动组表现 - 对照组表现

对照组要求：
- 特征相似（用户画像、消费习惯）
- 未受活动影响
- 样本量足够
```

#### 方法三：增量模型
```
# 分层增量计算
总增量 = Σ(各分层活动期 - 各分层基准期)

# 考虑自然增长
净增量 = 活动期 - 基准期 × (1 + 自然增长率)
```

### 4.3 增量评估注意事项

```
常见陷阱：
├── 把全部销量归因于活动
│   └── 正确：只认活动直接带来的销量
│
├── 忽略季节性因素
│   └── 正确：与去年同期对比
│
├── 忽略活动叠加效应
│   └── 正确：单独评估每个活动
│
└── 忽略用户自然消费
│   └── 正确：计算增量而非全部
```

## 五、活动ROI计算

### 5.1 ROI公式

```
基础ROI = 活动带来GMV / 活动投入成本

增量ROI = 活动增量GMV / 活动投入成本

考虑毛利的ROI：
ROI = (活动增量GMV × 毛利率) / 活动投入成本
```

### 5.2 成本构成

```
活动成本构成：
├── 直接成本
│   ├── 优惠让利金额
│   ├── 赠品成本
│   └── 物流成本增量
│
├── 推广成本
│   ├── 广告投放费用
│   ├── 渠道推广费用
│   └── KOL合作费用
│
└── 运营成本
    ├── 人力成本
    ├── 系统开发成本
    └── 客服成本增量
```

### 5.3 收益计算

```
活动收益构成：
├── 直接收益
│   ├── 活动期GMV
│   └── 活动增量GMV
│
├── 间接收益
│   ├── 新客获取
│   ├── 品牌曝光
│   └── 用户激活
│
└── 长期收益
    ├── 用户LTV提升
    ├── 复购率提升
    └── 口碑传播
```

## 九、注意事项

1. **增量思维**：活动评估的核心是增量，不是全部
2. **提前规划**：活动前明确目标、定义增量、确定对照
3. **认链接不认说法**：只认活动直接带来的效果
4. **全面成本**：ROI计算要考虑全部成本
5. **经验沉淀**：每次活动都要总结可复用的经验
