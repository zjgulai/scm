## 五、交叉线节点

| 交叉线 | 节点 | 上游产出（输入） | 产出 Agent | 下游产出（输出） | 消费 Agent | 格式 |
|--------|------|------------------|------------|------------------|------------|------|
| 交叉线1 | VOC→订单 | 痛点/主题表、建议卖点方向（表：theme_code, theme_name, channel_id, country_code, spu_id, priority） | 订单 Agent | 建议组合清单、建议卖点清单 | 产品/运营 | 表/清单 |
| 交叉线2 | 社媒VOC→营销ROI | 高浓度社区/话题表（community_name, platform, country_scope, topic_tag, priority）；专题④ 活动与 ROI | 营销 Agent | 高浓度社区清单、建议投放形式+ROI | 投放/运营 | 表+清单 |
| 交叉线3a | 退款→VOC | 售后/退款主题输入表（order_id, return_id, return_reason_code, is_partial_return, theme_suggested, country_code, channel_id） | 订单 Agent 产出 → VOC Agent 消费 | 售后/退款主题 VOC 摘要、双重视图结论 | 客服/产品 | 表+文本 |
| 交叉线3b | 退款→渠道 | 退款率/异常国家渠道 | 订单 Agent 产出 → 渠道 Agent 消费 | 渠道健康度与风险预警（纳入退款） | 战略解码 | 表 |
| 交叉线4a | 渠道→VOC | 国家×渠道画像、生命周期、流量结构（country_code, channel_id, lifecycle_stage, gmv, margin_pct, risk_flag） | 渠道 Agent 产出 → VOC Agent 消费 | VOC 分层分析规则（按战略角色/流量结构） | VOC Agent、营销 Agent | 文档/配置 |
| 交叉线4b | 渠道+营销→投放 | 国家×渠道画像、活动类型 ROI | 渠道+营销 Agent | 投放优先级建议（加码/守住/收缩） | 营销/运营 | 清单 |
| 交叉线4c | 营销→渠道 | 活动类型 ROI、精细化 ROI | 营销 Agent 产出 → 渠道 Agent 消费 | 渠道策略与费用校准（可选） | 渠道 Agent | 表/文本 |
