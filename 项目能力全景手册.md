# 项目能力全景手册：能解决什么问题、提供什么功能、给出什么指引

> **项目定位**：面向 60-80 亿规模母婴出海企业（Momcozy）的「数智决策-AI 效能作战室」
> **数据源**：40 个跨境电商分析 Skills、20 个麦肯锡 PPT Skills、7 个业务专项 Skills、34 个工作流 Skills、4 大专题×14 子课题×4 交叉故事线、供应链全链路优化子项目、BI 数据字典、外部数据研究框架

---

## 一、能回答的业务问题（按角色）

### 1.1 管理层（CEO / GM / VP）

| 问题 | 对应能力 | 产出形式 |
|------|---------|---------|
| 各国家×渠道的规模、增速、利润结构如何？哪些该加码、哪些该收缩？ | 专题③ 渠道生命周期 + 国家画像 | 管理层 PPT + 决策矩阵 |
| 全域 VOC 声量趋势如何？用户对我们的评价 vs 竞品差在哪？ | 专题① 全域 VOC 数据洞察 | VOC 看板 + 竞品对比表 |
| 营销花了多少钱、带来了多少增量？各类活动的 ROI 基准是什么？ | 专题④ 营销项目类型 ROI 量化 | ROI 基准表 + 精细化 ROI 表 |
| 订单履约链路中哪里是瓶颈？退款是策略问题还是产品问题？ | 专题② 订单数量与质量提升 | 节点诊断表 + 退款归因表 |
| 供应链全链路成本效率如何？库存健康度怎样？ | SCM 子项目 | 供应链洞察报告 + 管理层摘要 |
| 月度目标如何 SMART 化？KPI 如何与专题产出对齐？ | kpi-smart-monthly-goal | KPI 制定表格 |

### 1.2 分析岗（数据分析师 / BI）

| 问题 | 对应能力 | 产出形式 |
|------|---------|---------|
| 毛利率下降是哪个因素贡献最大？以价换量还是成本上升？ | margin-attribution / momcozy-margin-analysis | 因素分解瀑布图 + 归因表 |
| 销售额增长是量的驱动还是价的驱动？哪个品线/渠道贡献最大？ | contribution-calculation / complex-contribution | 贡献度分解表 |
| 同比/环比变化的驱动因素拆解（偏微分/全微分） | complex-contribution | 弹性系数 + 驱动项分解 |
| 用户分群怎么做？RFM 分层后各群特征是什么？ | customer-segmentation | 用户分群表 + 群画像 |
| 留存率如何？不同群组的生命周期价值差异？ | cohort-analysis | 群组留存矩阵 + LTV 估算 |
| 转化漏斗中哪一步流失最大？ | conversion-funnel | 漏斗图 + 流失分析 |
| 如何设计指标体系？北极星指标到 L0-L3 怎么拆？ | metrics-system-design / supply-chain-indicator-decoding | 指标树 + 责任人映射 |

### 1.3 运营岗（渠道/国家/产品/客服）

| 问题 | 对应能力 | 产出形式 |
|------|---------|---------|
| 各渠道的效果如何？自然流 vs 付费广告 vs 红人转化差异？ | channel-effect-analysis / traffic-source-analysis | 渠道效果对比表 + 流量结构表 |
| 哪些商品经常一起买？怎么设计搭售组合？ | basket-analysis / momcozy-product-line-basket-reporting | 关联规则表 + 搭售策略 |
| 某单品/品类的 SPU 如何优化？下钻归因到哪个 SKU？ | 专题04 下钻归因 | TOP10 瀑布图 + SPU 洞察 |
| 促销活动的效果如何？ROI 是否达标？ | promotion-analysis / sales-conversion-analysis | 活动效果表 |
| 用户评论中的主要痛点和亮点是什么？ | voc-insights / social-media-analysis | 痛点亮点清单 + 标签分布 |
| 本土竞品如何被评价？哪些卖点更打动当地用户？ | competitive-analysis | 竞品对比表 + 本土化话术 |

### 1.4 营销岗（投放/品牌/增长）

| 问题 | 对应能力 | 产出形式 |
|------|---------|---------|
| 广告投放的 ROAS 如何？哪个渠道效率最高？ | ad-attribution / ad-performance-analysis | 渠道 ROAS 对比表 |
| 如何量化营销活动带来的销售额增量？ | 专题④ 曝光归因模型 | 归因模型 + ROI 表 |
| 用户 LTV 如何？各活动类型对 LTV 的增量贡献？ | LTV 分段 + 增量分析 | LTV 增量表 |
| 社媒 VOC 中哪些社区/话题目标客群浓度高？ | 交叉线2：社媒 VOC → 垂类投放 | 高浓度社区清单 + 投放建议 |
| 内容/社媒的声量与互动趋势？ | social-media-analysis | 声量趋势图 |

## 五、智能体与自动化能力

### 5.1 Data Agent 架构

```
用户自然语言输入
  → agent/intent_parser.py（意图解析：15 种意图模式匹配）
  → agent/skills_router.py（Skill 路由：40 Skills 索引 + 关键词降级）
  → engine/data_processor.py（数据处理：委托 12 条分析管道）
  → output/ppt_generator.py（PPT 生成：真正的 .pptx）
  → output/chart_engine.py（图表生成：PNG 瀑布图/柱状图/折线图/饼图）
  → output/report_assembler.py（报告组装：Markdown + JSON）
```
