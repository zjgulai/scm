# 积加 SCM 帮助文档来源地图

## 证据边界

本文件记录本轮已阅读和用于归纳的公开帮助文档。公开帮助文档能支撑业务流程、字段说明、部分公式和状态流转，但不能证明公司内部实际配置。

以下结论属于文档级证据，不等同于企业实际数据源事实：

- 某字段在帮助文档中出现。
- 某字段在帮助文档中被解释。
- 某公式在帮助文档中被说明。
- 某模块之间存在单据关联。

以下内容必须通过企业实际 ERP 页面、导出、接口或数据库复核：

- 字段是否对本公司可见。
- 字段是否因权限被隐藏。
- 参数是否开启。
- 指标映射是否被二次配置。
- 导出字段名是否与文档字段一致。
- 实际业务是否绕过 ERP 单据。
