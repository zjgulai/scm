# 备货库存平台规则字典

## 产品库存

| 字段 | 规则 | 粒度 |
|---|---|---|
| 计划库存 | SRM 需求池待订单状态 + SRM 采购订单新建、变更中、待供应商确认、供应商拒绝、易仓处理中状态 | 仓库 + 供应链 SKU |
| 采购未交库存 | 采购订单明细和送货单明细组合，采购订单状态已确认；送货单状态新建、已拒绝、待计划确认；过滤 PO 已关闭 | 仓库 + 供应链 SKU |
| 可用库存 | 直接取积加可用量 | 仓库 + 供应链 SKU |
| 预占库存 | 直接取积加已处理预占 | 仓库 + 供应链 SKU |
| 冻结库存 | 默认为 0 | 仓库 + 供应链 SKU |
| 在库库存 | 可用 + 预占 + 冻结 + 不良品 | 仓库 + 供应链 SKU |
| 在库良品库存 | 可用 + 预占 + 冻结 | 仓库 + 供应链 SKU |
| 不良品库存 | 直接取积加次品量 | 仓库 + 供应链 SKU |

## FBA 库存

| 字段 | 规则 | 粒度 |
|---|---|---|
| 计划库存 | 调拨单待拣货、待出库状态，筛选调入仓为 Amazon 平台仓，按 MSKU + 仓库汇总调拨量 | 仓库 + MSKU |
| 在途库存 | FBA 货件状态为 WORKING、SHIPPED、IN_TRANSIT、DELIVERED、CHECKED_IN、READY_TO_SHIP，且关联调拨单为调拨在途或已完成；在途数量等于发货量 | 仓库 + MSKU |
| 可用库存 | 本地可售 + 预留-运营中心转运 + 预留-运营中心正在处理 - 预留-买家订单 | 仓库 + MSKU |
| 预占库存 | 预留-买家订单 | 仓库 + MSKU |
| 冻结库存 | 预留-运营中心转运 + 预留-运营中心正在处理 | 仓库 + MSKU |
| 在库库存 | 本地可售 + 预留-运营中心转运 + 预留-运营中心正在处理 + 预留-买家订单 + 不可售 | 仓库 + MSKU |
| 在库良品库存 | 本地可售 + 预留-运营中心转运 + 预留-运营中心正在处理 | 仓库 + MSKU |
| 不良品库存 | 不可售字段 | 仓库 + MSKU |

## Shopify/独立站库存

| 字段 | 规则 | 粒度 |
|---|---|---|
| 计划库存 | 调拨单待拣货、待出库状态，筛选调入仓库归属为 Shopify，按供应链 SKU + 仓库汇总 | 供应链 SKU + 仓库 |
| 在途库存 | 调拨单为调拨在途或已完成，关联发货单状态为提货中、已出运、入库中，按发货量汇总 | 供应链 SKU + 仓库 |
| 可用库存 | 取积加库存同步三方仓库存可用量 | 供应链 SKU + 仓库 |
| 预占库存 | 取积加库存同步三方仓库存预占量 | 供应链 SKU + 仓库 |
| 冻结库存 | 默认为 0 | 供应链 SKU + 仓库 |
| 不良品库存 | 取积加库存同步三方仓库存次品量 | 供应链 SKU + 仓库 |

## Walmart 库存

| 字段 | 规则 | 粒度 |
|---|---|---|
| 计划库存 | 调拨单待拣货、待出库状态；调入仓归属 Walmart 且国家 US；用 FNSKU 查询 GTIN，再按 GTIN 和 ItemID 映射汇总 | ItemID + 国家 |
| 在途库存 | 物流跟踪表状态为正常在途、出口查验未放行、进口查验未放行、延误风险；渠道 Walmart，国家 US，数量等于 SKU 明细数量 | ItemID + 国家 |
| 可用库存 | 积加平台仓库存 Walmart 可售字段 | ItemID + 国家 |
| 预占库存 | 默认为 0 | ItemID + 国家 |
| 冻结库存 | 默认为 0 | ItemID + 国家 |
| 在库库存 | 可用 + 预占 + 冻结 + 不良品 | ItemID + 国家 |
| 在库良品库存 | 可用 + 预占 + 冻结 | ItemID + 国家 |
| 不良品库存 | 积加平台仓库存 Walmart 拒收损坏字段 | ItemID + 国家 |

## TikTok 库存

| 字段 | 规则 | 粒度 |
|---|---|---|
| 计划库存 | 调拨单待拣货、待出库状态；调入仓归属 TikTok、仓库类型海外仓、区分国家；按销售映射映射到 MSKU 汇总 | MSKU + 国家 |
| 在途库存 | 物流跟踪表状态为正常在途、出口查验未放行、进口查验未放行、延误风险；渠道 TikTok，按目的国家汇总 SKU 明细数量 | MSKU + 国家 |
| 可用库存 | FBT 库存可用量 + 积加三方仓库存按 TikTok 海外仓和销售映射汇总的可用量 | MSKU + 国家 |
| 预占库存 | FBT 库存预占量 + 积加三方仓库存按 TikTok 海外仓和销售映射汇总的预占量 | MSKU + 国家 |
| 冻结库存 | 默认为 0 | MSKU + 国家 |
| 在库库存 | 可用 + 预占 + 冻结 + 不良品 | MSKU + 国家 |
| 在库良品库存 | 可用 + 预占 + 冻结 | MSKU + 国家 |
| 不良品库存 | FBT 次品量 + 积加三方仓库存按 TikTok 海外仓和销售映射汇总的次品量 | MSKU + 国家 |

## 页面交互规则

| 功能 | 规则 |
|---|---|
| 单条搜索 | 默认按平台主键搜索，支持模糊搜索 |
| 批量搜索 | 一行一项，最多 100 行，重复数据按一条，精确搜索 |
| 产品名称搜索 | 单条搜索，支持模糊搜索 |
| 导出 | 异步导出，导出字段同页面字段 |
| 排序/筛选 | 库存字段支持筛选，部分页面规则要求所有库存字段可排序 |
