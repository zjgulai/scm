# 数据能力参考目录

本目录用于**知识萃取与Skill创建**的参考资料库。

## 目录结构

- `0401-数据分析专业知识/` - 数据分析专业理论知识
- `0402-运营数据分析/` - 运营数据分析实战资料

## 用途

作为 `knowledge_exploration.py`、`biji_knowledge.py`、`quick_exploration.py` 等知识萃取工具的输入源，用于：

1. 从外部知识库中萃取分析方法和业务洞察
2. 创建新的专业Skills
3. 丰富现有Skills的理论基础

## 相关工具

- `biji_knowledge.py` - 知识库读取与萃取
- `knowledge_exploration.py` - 知识探索与关联
- `quick_exploration.py` - 快速知识查询
