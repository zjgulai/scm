# 积加仓库模块业务逻辑与指标血缘

## 模块定位

仓库模块负责回答四个问题：

1. 货在哪里。
2. 货是什么状态。
3. 货为什么增加或减少。
4. 货能否被销售、调拨、装配或补货计算使用。

仓库模块不能只看库存汇总。正确顺序是：单据产生库存变化，库存流水记录变化，库存快照承载当前结果，批次库存支持实物追溯。

## 核心子模块

| 子模块 | 业务作用 | 关键对象 |
|---|---|---|
| 产品库存（NEW） | SKU/MSKU/仓库/站点维度库存面板 | 计划量、采购量、交货量、在途、良品、预占、可用 |
| 仓库库存 | 仓库维度库存汇总 | 仓库汇总指标 |
| 批次库存 | 实物批次查询和追溯 | 实物编码、初始入库单号、批次日志 |
| 库存流水 | 实时记录出入库和数量变化 | 操作类型、关联单据、良品变化、次品变化 |
| 入库单 | 自营仓/供应商仓收货入库 | 应收货、已收货、已入库、入库状态 |
| 质检单 | 工厂或仓库质检 | 质检结果、良品、次品 |
| 调拨单 | 仓间库存移动 | 调拨量、已出运、已调入 |
| 调整单 | 同仓不同站点库存调整 | 调出站点、调入站点、调整数量 |

## 业务逻辑

### 产品库存（NEW）

产品库存是库存指标体系的核心展示层。它不是单一来源表，而是采购计划、装配计划、采购订单、交货单、发货单、入库单、平台仓库存报告等多来源汇总后的结果。

文档显示新版产品库存强调：

- 平台仓库存指标可通过参数配置映射。
- 库存周转指标所需库存字段可以自定义配置。
- 采购计划量和装配计划量被拆得更细。
- 非平台仓 7 天日均消耗口径更偏向订单出库。

这意味着产品库存字段必须记录：

- 口径版本。
- 参数配置。
- 来源系统。
- 是否平台仓。
- 是否共享库存或锁站点库存。

### 仓库库存

仓库库存是按仓库聚合的总览，不提供操作。它适合作为核对和看板入口，但不适合作为底层事实表。底层指标仍应回到产品库存明细、库存流水和源单据。

### 批次库存

批次库存用于实物追溯。初始数据来自系统初始化和仓库入库，后续状态变化持续跟踪。

适用场景：

- 追踪某个订单发出的实物批次。
- 查看组合品装配后是否生成批次记录。
- 追查退货回库实物状态。

不适用场景：

- 直接计算经营层库存总量。
- 替代产品库存快照。

### 库存流水

库存流水实时生成，记录自营仓、供应商仓、三方仓库存出入库情况。它是解释库存变化的关键事实来源。

字段结构应分为：

- 流水基本信息：发生时间、操作类型、操作对象、关联单据。
- 数量变化信息：良品量变化、次品量变化。

### 入库单

入库单覆盖采购或调拨到自营仓、海外自营仓、供应商仓的收货入库流程。

关键状态：

- 待入库。
- 部分入库。
- 已完成。
- 已取消。

关键数量：

- 应收货。
- 已收货。
- 已入库。
- 良品。
- 次品。
- 退货。

### 调拨单

调拨单支持从自营仓、供应商仓、三方仓调出，调入其他仓或平台仓。

关键规则：

- 查看调拨单需同时拥有调出仓和调入仓权限。
- 调入自营仓或供应商仓时，已调入取入库单已入库量。
- 调入三方仓时，已调入取三方仓接口同步入库数。
- 调入平台仓时，调拨单已调入可能不直接体现，需要看发货单或平台报告。

### 调整单

调整单用于国内仓同一仓库内不同站点之间的备货库存调整。它本质上不是实物跨仓移动，而是站点归属或共享/锁定关系的调整。

## 指标字典

| 指标编码 | 指标 | 粒度 | 口径 |
|---|---|---|---|
| SCM-WH-001 | 采购计划量 | SKU/MSKU + 仓库 + 站点 | 未分单采购计划数量 |
| SCM-WH-002 | 装配计划量 | SKU/MSKU + 仓库 + 站点 | 预装需求或装配单未完成数量 |
| SCM-WH-003 | 计划量 | SKU/MSKU + 仓库 + 站点 | 采购计划量 + 装配计划量 |
| SCM-WH-004 | 采购量 | SKU/MSKU + 仓库 + 站点 | 已生成采购订单但未完成后续交货的数量 |
| SCM-WH-005 | 交货量 | SKU/MSKU + 仓库 + 站点 | 采购订单已进入交货单的数量 |
| SCM-WH-006 | 已下单量 | SKU/MSKU + 仓库 + 站点 | 计划量、采购量、交货量等未入库供应链数量组合 |
| SCM-WH-007 | 待出运量 | SKU/MSKU + 仓库 + 站点 | 已绑定发货相关流程但未出运的数量 |
| SCM-WH-008 | 在途量 | SKU/MSKU + 仓库 + 站点 | 已发货和正在接收等运输中数量 |
| SCM-WH-009 | 应收货 | 入库单 + SKU | 创建入库单时来源发货单对应数量 |
| SCM-WH-010 | 已收货 | 入库单 + SKU | 仓库实际收货数量 |
| SCM-WH-011 | 已入库 | 入库单 + SKU | 仓库确认入库数量 |
| SCM-WH-012 | 良品量 | SKU/MSKU + 仓库 + 站点 | 可作为良品库存管理的实物数量 |
| SCM-WH-013 | 次品量 | SKU/MSKU + 仓库 + 站点 | 质检或库存状态为次品的数量 |
| SCM-WH-014 | 预占量 | SKU/MSKU + 仓库 + 站点 | 被订单、转运或处理流程占用的数量 |
| SCM-WH-015 | 良品可用量 | SKU/MSKU + 仓库 + 站点 | 良品量扣减预占等不可用部分后的可用数量 |
| SCM-WH-016 | 调拨量 | 调拨单 + SKU | 调拨单产品总数量 |
| SCM-WH-017 | 已出运 | 调拨单/发货单 + SKU | 已关联发货单且已出运数量 |
| SCM-WH-018 | 已调入 | 调拨单 + SKU | 按调入仓类型取入库单或三方仓接口已入库量 |
| SCM-WH-019 | 批次库存量 | 批次 + SKU + 仓库 | 某实物批次当前数量或状态 |
| SCM-WH-020 | 库存流水变化量 | 流水 + SKU | 单次操作导致的良品或次品变化 |

## 指标血缘

```text
采购计划/装配计划/采购订单/交货单/发货单/调拨单
  -> 入库单/出库单/调整单/盘点单
  -> 库存流水
  -> 产品库存快照
  -> 仓库库存汇总
  -> 智能补货和 BI 库存指标
```

上游数据：

- 采购计划。
- 采购订单。
- 交货单。
- 发货单。
- 调拨单。
- 入库单。
- 质检单。
- 盘点、调整、状态变更单。
- 平台仓库存报告。

下游数据：

- 产品库存快照。
- 仓库库存汇总。
- 批次库存。
- 智能补货。
- 库龄和周转分析。
- 库存异常和对账。

## 数据建模建议

| 表 | 粒度 | 必备字段 |
|---|---|---|
| `fact_inventory_snapshot` | 快照时间 + SKU/MSKU + 仓库 + 站点 | 计划、采购、交货、在途、良品、次品、预占、可用 |
| `fact_inventory_transaction` | 流水单行 | 操作类型、来源单据、良品变化、次品变化、发生时间 |
| `fact_inbound_order_line` | 入库单 + SKU | 应收货、已收货、已入库、良品、次品、入库方式 |
| `fact_transfer_order_line` | 调拨单 + SKU | 调出仓、调入仓、调拨量、已出运、已调入 |
| `fact_batch_inventory_snapshot` | 快照时间 + 批次 + SKU | 初始入库单、实物状态、批次日志 |
| `meta_inventory_mapping` | 参数版本 + 仓库类型 | 平台仓库存指标映射、周转指标字段配置 |

## 后续核验清单

- 产品库存（NEW）导出字段完整清单。
- 平台仓库存指标映射配置。
- 库存周转字段配置。
- 良品可用量、可调拨量、预占量在公司 ERP 中的实际公式。
- 调入平台仓的已调入数据是否从平台报告回写。
- 库存流水是否覆盖三方仓接口变化。
