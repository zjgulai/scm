# 积加 SCM 指标字典

## 使用规则

本指标字典是第一版草稿。字段名以帮助文档口径为基础，后续必须用公司 ERP 导出/API 字段复核。

每个指标上线前必须补齐：

- 指标负责人。
- 实际来源表或导出文件。
- 实际字段名。
- 计算 SQL。
- 数据质量规则。
- 刷新频率。

## 计划指标

| 指标编码 | 指标名称 | 粒度 | 指标类型 | 计算/口径 | 主要来源 |
|---|---|---|---|---|---|
| SCM-PLAN-001 | 日均消耗量 | MSKU + 站点 + 日期 | 计算指标 | 历史订单或目标销量规则计算，可结合多周期权重 | 智能补货 |
| SCM-PLAN-002 | 目标库存天数 | MSKU + 目的仓 | 计算指标 | 计划审批、采购交期、质检、物流、入库处理、安全库存、补货周期等周期组合 | 智能补货 |
| SCM-PLAN-003 | 目标库存量 | MSKU + 目的仓 | 计算指标 | 目标库存天数 × 日均消耗量 | 智能补货 |
| SCM-PLAN-004 | 推荐采购量 | SKU/MSKU + 目的仓 | 决策指标 | 目标库存量扣减可用库存、可调拨、已下单、计划入库、待出运、在途等 | 智能补货 |
| SCM-PLAN-005 | 推荐调拨量 | SKU/MSKU + 调出仓 + 目的仓 | 决策指标 | 目标库存量扣减目的仓库存和可覆盖在途后的调拨需求 | 智能补货 |

| SCM-PLAN-007 | 缺货天数 | MSKU + 目的仓 | 预警指标 | 目标到货日期内未被补上的缺货天数 | 红绿线 |
| SCM-PLAN-008 | 缺货量 | MSKU + 目的仓 | 预警指标 | 缺货期间目标销量扣减可用库存和计划入库后的缺口 | 红绿线 |
| SCM-PLAN-009 | 建议发货量 | MSKU + 发货点 | 决策指标 | 连续补货规则下生成的发货数量 | 连续补货建议 |

## 统一口径风险

| 风险 | 影响指标 | 处理方式 |
|---|---|---|
| 智能补货参数不同 | 推荐采购量、目标库存量、缺货日期 | 建参数快照表 |
| FBA 在途来源不同 | 在途量、推荐采购量 | 区分 ERP 单据在途和平台报告在途 |
| 平台仓库存映射可配置 | 良品量、可用量、周转 | 建库存映射配置表 |
| 已作废单据混入 | 计划量、采购量、交货量 | 状态过滤规则必须固化 |
| 物流实际费用跨月 | 成本、利润、估实差异 | 独立建估实差异事实表 |
| 供应商仓货权延迟转移 | 应付、对账、货权库存 | 单独建货权转移事实表 |
