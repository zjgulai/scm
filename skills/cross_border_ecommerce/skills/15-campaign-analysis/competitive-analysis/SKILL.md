# 跨境电商竞品分析

## 一、Skill概述

竞品分析是了解市场竞争格局、制定竞争策略的重要方法。通过系统化分析竞争对手，可以发现市场机会、优化产品策略、提升竞争优势。

**核心价值**：
- 了解市场竞争格局
- 发现差异化机会
- 优化产品/定价策略
- 制定竞争策略

---

## 二、竞品分析框架

### 2.1 分析框架总览

```
竞品分析五步法
├── Step1: 确定分析目的
│   └── 为什么要做竞品分析？解决什么问题？
│
├── Step2: 选择竞品对象
│   └── 谁是竞争对手？直接/间接/潜在
│
├── Step3: 收集竞品信息
│   └── 竞品信息从哪来？公开/购买/调研
│
├── Step4: 对比分析
│   └── 我们和竞品差距在哪？优势/劣势
│
└── Step5: 制定策略
    └── 如何应对竞争？进攻/防守/差异化
```

### 2.2 竞品分类

```
竞品分类
├── 直接竞品
│   ├── 同品类、同市场
│   ├── 满足相同用户需求
│   └── 例：同品类的其他卖家
│
├── 间接竞品
│   ├── 不同品类、相同需求
│   ├── 替代性产品
│   └── 例：母婴品类中，纸尿裤vs布尿裤
│
└── 潜在竞品
    ├── 可能进入市场的玩家
    ├── 新技术/新模式威胁
    └── 例：国内品牌出海
```

## 四、对比分析方法

### 4.1 产品对比矩阵

```
产品对比维度
┌─────────────┬──────────┬──────────┬──────────┐
│   维度      │  我们    │  竞品A   │  竞品B   │
├─────────────┼──────────┼──────────┼──────────┤
│ 价格        │          │          │          │
│ 功能        │          │          │          │
│ 质量        │          │          │          │
│ 设计        │          │          │          │
│ 评价        │          │          │          │
│ 销量        │          │          │          │
└─────────────┴──────────┴──────────┴──────────┘

评分标准：
5分 = 远超竞品
4分 = 略优于竞品
3分 = 与竞品持平
2分 = 略逊于竞品
1分 = 远逊于竞品
```

### 4.2 SWOT分析

```
SWOT分析框架
┌─────────────────────────┬─────────────────────────┐
│        优势(S)          │        劣势(W)          │
├─────────────────────────┼─────────────────────────┤
│ - 比竞品好的地方        │ - 比竞品差的地方        │
│ - 核心竞争力            │ - 需要改进的方面        │
└─────────────────────────┴─────────────────────────┘
┌─────────────────────────┬─────────────────────────┐
│        机会(O)          │        威胁(T)          │
├─────────────────────────┼─────────────────────────┤
│ - 市场机会              │ - 市场威胁              │
│ - 可利用的趋势          │ - 需要防范的风险        │
└─────────────────────────┴─────────────────────────┘

策略制定：
- SO策略：利用优势抓住机会
- WO策略：克服劣势抓住机会
- ST策略：利用优势应对威胁
- WT策略：克服劣势应对威胁
```

### 4.3 定位图分析

```
价格-质量定位图
                    高质量
                      │
         竞品A        │        我们
                      │
    ──────────────────┼──────────────────
         竞品B        │        竞品C
                      │
                    低质量
         低价                   高价

分析要点：
- 各竞品的市场定位
- 市场空白区域
- 我们的定位选择
```

## 六、竞争策略制定

### 6.1 竞争策略类型

```
竞争策略选择
├── 进攻策略
│   ├── 正面进攻：价格战、广告战
│   ├── 侧翼进攻：细分市场、差异化
│   └── 包围进攻：多产品、多渠道
│
├── 防守策略
│   ├── 阵地防守：巩固现有优势
│   ├── 侧翼防守：保护薄弱环节
│   └── 先发防守：提前布局
│
└── 差异化策略
    ├── 产品差异化
    ├── 服务差异化
    ├── 品牌差异化
    └── 渠道差异化
```

### 6.2 策略选择矩阵

```
           市场份额高
               │
    防守策略   │   扩张策略
               │
───────────────┼───────────────
               │
    撤退策略   │   进攻策略
               │
           市场份额低

    竞争力弱          竞争力强
```

## 九、注意事项

1. **明确目的**：竞品分析要服务于具体决策
2. **选对竞品**：不是所有同行都是竞品
3. **信息可靠**：确保信息来源可靠、数据准确
4. **客观分析**：避免主观偏见，用数据说话
5. **持续监控**：市场在变，竞品分析需要持续进行
