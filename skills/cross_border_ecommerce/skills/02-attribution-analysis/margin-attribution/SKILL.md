# 跨境电商毛利率多维归因分析 Skill

## 概述

毛利率是跨境电商盈利能力的核心指标。本Skill提供系统化的毛利率归因分析方法，帮助用户：

- **量化各因素贡献**: 明确各维度对毛利率变动的具体影响
- **区分变动来源**: 分离结构变动（运营策略）与自身变动（成本/定价）
- **定位问题区间**: 通过订单分箱识别高/低贡献区间
- **支持决策优化**: 基于归因结果制定针对性改进策略

### 核心公式

**毛利率基础公式**:
```
毛利率(GPM) = 毛利额 / 销售额 = 1 - 成本率

运营毛利率(Operating GPM) = 1 - 成本率(PCR) - 广告费率(MCR) - 履约费率(FCR)
```

**变动贡献公式**:
```
ΔGPM_op ≈ -ΔPCR - ΔMCR - ΔFCR
```

## 核心功能

### 1. 静态贡献分析

**适用场景**: 分析单周期内各区间对整体毛利率的贡献占比

**计算公式**:
```
单个区间毛利率: G_i = P_i / S_i

单个区间对整体毛利率的贡献:
Contribution_i = (S_i / S_总) × G_i = w_i × G_i

单个区间贡献占比:
Contribution_ratio_i = Contribution_i / G_总 × 100%
```

**参数说明**:
| 参数 | 说明 |
|------|------|
| S_i | 第i个区间的总销售额 |
| P_i | 第i个区间的总毛利 |
| G_i | 第i个区间的毛利率 |
| w_i | 第i个区间销售额占总销售额的比例 |
| G_总 | 整体毛利率 |

**验证条件**: 所有区间贡献占比加总 = 100%

### 2. 动态变动贡献分析

**适用场景**: 分析本期对比上期，整体毛利率变动时各区间贡献

**计算公式**:
```
整体毛利率变动: ΔG = G_本期 - G_上期

单个区间i对整体毛利率变动的总贡献:
Contribution_Δi = Δw_i × G_i上期 + w_i本期 × ΔG_i

其中:
- Δw_i = w_i本期 - w_i上期 (销售额占比变动)
- ΔG_i = G_i本期 - G_i上期 (区间自身毛利率变动)

贡献占比:
Contribution_ratio_Δi = Contribution_Δi / ΔG × 100%
```

### 3. 结构变动 vs 自身变动分解

**核心逻辑**: 将总变动贡献拆分为两部分，业务含义完全不同

**结构变动贡献** = Δw_i × G_i上期

> 代表「订单占比变化」带来的影响，对应运营端动作结果
> （如高客单占比提升、低客单引流变多都属于结构变化）

**自身毛利率变动贡献** = w_i本期 × ΔG_i

> 代表「同区间内毛利率变化」带来的影响，对应成本/定价端变化
> （如进货涨价、降价促销、商品结构调整）

**判断逻辑**:
```
如果「结构变动贡献」占比大 → 调整运营策略（改引流方向、做凑单/组合推荐）

如果「自身毛利率变动贡献」占比大 → 从成本定价端优化
    （换供应商、调整折扣力度、换区间内的商品）
```

### 4. 订单分箱分析

#### 4.1 客单价分箱

将订单按客单价划分为5个区间，分析各区间贡献：

| 客单价区间 | 分箱逻辑 | 业务含义 |
|-----------|---------|---------|
| 高 | >150元 | 高价值客户，贡献主要利润 |
| 中高 | 90-150元 | 核心客户群 |
| 一般 | 50-90元 | 普通客户 |
| 低 | 20-50元 | 引流客户 |
| 极低 | ≤20元 | 价格敏感客户 |

**分箱策略建议**:
- **等频分箱**: 每个区间订单量相近，适合偏态分布（推荐）
- **等距分箱**: 区间宽度相等，适合均匀分布

#### 4.2 客品数分箱

将订单按客品数（单订单商品数）划分为5个区间：

| 客品数区间 | 分箱逻辑 | 业务含义 |
|-----------|---------|---------|
| 高 | ≥8件/单 | 囤货型客户，高毛利 |
| 中高 | 5-7件/单 | 组合购买型 |
| 一般 | 3-4件/单 | 普通购买 |
| 低 | 2件/单 | 凑单型 |
| 极低 | 1件/单 | 零买型，低毛利 |

### 5. 运营维度拆解

**运营成本侧拆解**:
```
品线毛利率构成:
GPM_op = 1 - 成本率(PCR) - 广告费率(MCR) - 履约费率(FCR)

变化贡献度拆解:
ΔGPM_op ≈ -ΔPCR - ΔMCR - ΔFCR
```

**销售订单侧拆解**:
```
毛利率 = 毛利额 / 销售额
       = 单均毛利额 / 单均销售额
       = 订单价 × 单均毛利率 × 订单数 / 销售额
```

## 使用方法

### 输入数据要求

**基础数据格式**:
```csv
period,bin_name,sales,gross_profit,gross_margin
2024-01,高客单,2400000,912000,0.38
2024-01,中高客单,2600000,819000,0.315
2024-01,一般客单,2500000,300000,0.12
2024-01,低客单,1500000,111000,0.074
2024-01,极低客单,1000000,82000,0.082
```

**字段说明**:
| 字段 | 说明 | 示例 |
|------|------|------|
| period | 时间周期 | 2024-01, Q1, Week1 |
| bin_name | 分箱名称 | 高客单, 中高客单 |
| sales | 区间销售额 | 2400000 |
| gross_profit | 区间毛利 | 912000 |
| gross_margin | 区间毛利率 | 0.38 |

### 分析流程

```
步骤1: 数据加载与分箱
- 读取订单级数据
- 按客单价/客品数进行分箱
- 计算各区间基础指标

步骤2: 静态贡献计算
- 计算各区间销售额占比
- 计算各区间对整体毛利率贡献
- 识别高贡献区间

步骤3: 动态变动分析
- 计算两周期对比的变动
- 拆分结构变动与自身变动
- 识别主要变动来源

步骤4: 运营维度拆解
- 成本率变动分析
- 广告费率变动分析
- 履约费率变动分析

步骤5: 可视化与建议
- 生成贡献度瀑布图
- 输出改进建议
```

## 示例场景

### 示例1: 毛利率下滑归因

**场景**: 独立站毛利率同比下滑3.4pp，需要定位原因

**数据**:
| 指标 | MAT2025 P1 | MAT2026 P1 | 同比变动 |
|------|-----------|-----------|---------|
| 毛利率 | 35.4% | 32.0% | -3.4pp |
| GMV | $500K | $970K | +94% |
| 毛利额 | $177K | $310K | +75% |

**归因分析**:

**运营成本侧**:
```
成本率变动: +1.5pp → 贡献 -1.5%
广告费率变动: +1.0pp → 贡献 -1.0%
促销折扣变动: +0.8pp → 贡献 -0.8%
履约费率变动: -0.1pp → 贡献 +0.1%

合计: -3.2% ≈ -3.4%
```

**结论**:
1. 产品结构变化导致平均成本率上升（贡献-1.5%）
2. 广告费率攀升（贡献-1.0%）
3. 促销活动力度加大（贡献-0.8%）
4. 高毛利核心SPU销量占比下降是根本原因

**建议**:
- 优化产品组合，提升高毛利产品占比
- 重构促销策略，减少低效折扣
- 提升广告投放效率

## 注意事项

### 分箱策略选择

| 数据分布特征 | 推荐策略 | 原因 |
|------------|---------|------|
| 偏态分布（少数极高） | 等频分箱 | 避免某区间订单量过少 |
| 均匀分布 | 等距分箱 | 区间宽度一致，易于理解 |
| 双峰分布 | 自定义分箱 | 按业务含义划分 |

### 变动分析要点

1. **必须拆分结构变动与自身变动**: 才能精准定位变动原因
2. **关注贡献占比而非绝对值**: 识别关键驱动因素
3. **验证计算结果**: 贡献度之和应等于总变动

### 业务解读技巧

- **结构变动贡献大**: 调整运营策略（引流方向、凑单活动）
- **自身变动贡献大**: 优化成本定价端（供应商、折扣力度）
- **关注高贡献区间**: 优先保护和提升高贡献区间
